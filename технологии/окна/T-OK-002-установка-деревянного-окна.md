# Установка деревянного окна

> Монтаж окна из натуральной древесины с соблюдением принципа «изнутри плотнее, чем снаружи».

---

## Когда применять

- Деревянные дома
- Исторические здания
- Экологичный интерьер

## Когда НЕ применять

- Влажные помещения
- Помещения без отопления

---

## Необходимые материалы

| Материал | Количество | Примечание |
|----------|------------|------------|
| Окно деревянное | 1 шт | Сосна/дуб/лиственница |
| Монтажная пена | 1-2 баллона | Низкого расширения |
| ПСУЛ | По периметру | Снаружи |
| Пароизоляционная лента | По периметру | Изнутри |
| Антисептик | 0.5 л | Для торцов |

---

## Пошаговая инструкция

### Шаг 1: Подготовка
Снять створки, обработать торцы антисептиком, установить пластины, наклеить ПСУЛ.

### Шаг 2: Установка рамы
Установить на клинья, выставить по уровню, зазоры 15-25 мм.

### Шаг 3: Крепление
Закрепить пластинами, проверять уровень после каждого крепления.

### Шаг 4: Герметизация
Запенить на 2/3 глубины, наклеить пароизоляцию изнутри.

### Шаг 5: Сборка
Навесить створки, установить стеклопакеты, отрегулировать.

---

## Чек-лист качества

- [ ] Рама по уровню (≤1.5 мм/м)
- [ ] ПСУЛ снаружи, пароизоляция изнутри
- [ ] Нет продувания
