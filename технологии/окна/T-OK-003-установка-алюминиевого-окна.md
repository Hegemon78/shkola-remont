# Установка алюминиевого окна

> Монтаж алюминиевых конструкций для остекления балконов, витражей, панорамного остекления.

---

## Когда применять

- Остекление балконов и лоджий
- Витражи большой площади
- Раздвижные системы

## Когда НЕ применять

- Стандартные проёмы — дороже ПВХ
- Холодный профиль в отапливаемых помещениях

---

## Необходимые материалы

| Материал | Количество | Примечание |
|----------|------------|------------|
| Алюминиевая конструкция | По проекту | Тёплый/холодный профиль |
| Монтажная пена | 2-3 баллона | Низкого расширения |
| Анкера рамные | 8-12 шт | 10×112 мм |
| Герметик силиконовый | 2 тубы | Только нейтральный! |

---

## Пошаговая инструкция

### Шаг 1: Подготовка
Снять створки, извлечь стеклопакеты, установить пластины.

### Шаг 2: Установка рамы
Выставить по уровню (лазерный для больших конструкций), расклинить.

### Шаг 3: Крепление
Закрепить анкерами, не перетягивать — алюминий мягкий.

### Шаг 4: Соединение секций
Стыковать через соединительный профиль, герметизировать.

### Шаг 5: Сборка
Установить створки и стеклопакеты, загерметизировать нейтральным силиконом.

**Важно:** Кислотный силикон вызывает коррозию алюминия!

---

## Чек-лист качества

- [ ] Рама по уровню
- [ ] Стыки секций герметичны
- [ ] Створки работают плавно
