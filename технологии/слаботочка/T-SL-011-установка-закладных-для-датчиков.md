# Установка закладных для датчиков

> Технология подготовки точек для установки датчиков умного дома: движения, протечки, дыма, открытия окон/дверей. Включает прокладку кабелей и установку закладных элементов на этапе черновых работ.

---

## Когда применять

- Интеграция системы умного дома
- Установка проводных датчиков охранной сигнализации
- Система защиты от протечек
- Автоматизация освещения (датчики движения)
- Противопожарная система

## Когда НЕ применять

- Беспроводные датчики (на батарейках) — закладные не нужны
- Чистовой ремонт без штробления
- Временные/арендные помещения

---

## Преимущества

- Скрытая проводка — эстетика
- Надёжное питание датчиков (не нужны батарейки)
- Стабильная связь (проводная надёжнее Wi-Fi)
- Интеграция с любыми системами автоматизации

## Недостатки

- Требует планирования заранее
- Нельзя изменить расположение после ремонта
- Дополнительные расходы на кабели
- Сложнее, чем беспроводные системы

## Подготовка

### Проверить перед началом:

- [ ] Есть проект расположения датчиков (согласован)
- [ ] Определён тип системы (проводная/беспроводная)
- [ ] Выбран тип кабеля под датчики
- [ ] Определено место контроллера/хаба
- [ ] Штробы готовы или будут делаться

### Подготовить рабочее место:

1. Изучить проект расположения датчиков
2. Определить маршруты кабелей
3. Подготовить инструменты и материалы
4. Разметить точки установки

---

## Пошаговая инструкция

### Шаг 1: Определение точек установки

**Описание:** Размечаем места для каждого типа датчиков.

**Как делать:**

**Датчики движения:**
- Высота: 2.2-2.5 м (угол потолок-стена)
- Расположение: угол комнаты с максимальным обзором
- Зона действия: 8-12 м (PIR-датчики)

**Датчики протечки:**
- Под раковиной, за унитазом, у стиральной машины
- У ванны/душевой кабины
- Под бойлером
- У батареи/конвектора

**Датчики открытия:**
- На раме окна/двери
- Высота: 1.5-1.8 м (на уровне ручки)

**Датчики дыма:**
- Потолок, в центре помещения
- Не ближе 50 см от стены

**На что обратить внимание:**
- Избегать прямых солнечных лучей (датчики движения)
- Не ставить над источниками тепла
- Учитывать зону действия датчика

**Типичные ошибки:**
- Датчик движения напротив окна — ложные срабатывания
- Датчик протечки далеко от источника — поздно сработает

**Время:** ~20-30 минут

---

### Шаг 2: Разметка трасс

**Описание:** Размечаем маршруты кабелей от датчиков к контроллеру.

**Как делать:**
1. Отметить место контроллера/хаба (у слаботочного щита)
2. Нарисовать маршруты от каждого датчика
3. Группировать датчики по зонам для экономии кабеля
4. Измерить длину каждого участка
5. Добавить 20% запас

**На что обратить внимание:**
- Трассы отдельно от силовых кабелей
- Радиус изгиба ≥ 30 мм
- Минимальная длина трасс

**Типичные ошибки:**
- Не учитывать запас — не хватит кабеля
- Длинные петли — лишний расход

**Время:** ~15-20 минут

---

### Шаг 3: Установка подрозетников

**Описание:** Устанавливаем подрозетники для скрытых датчиков.

**Как делать:**
1. Разметить центр подрозетника
2. Высверлить отверстие коронкой 68 мм
3. Очистить отверстие от пыли
4. Вставить подрозетник, проверить глубину
5. Закрепить на гипсовую смесь или шурупами (ГКЛ)
6. Выровнять по уровню

**На что обратить внимание:**
- Подрозетник заподлицо со стеной (или на 1-2 мм глубже)
- Отверстия для кабеля открыты
- Надёжная фиксация

**Типичные ошибки:**
- Подрозетник выступает — мешает датчику
- Криво установлен — датчик будет криво

**Время:** ~10-15 минут на точку

---

### Шаг 4: Прокладка кабелей

**Описание:** Прокладываем кабели от контроллера к датчикам.

**Как делать:**
1. Проложить гофру от точки к контроллеру
2. Закрепить клипсами с шагом 30-40 см
3. Протянуть кабель через гофру
4. Оставить запас у датчика: 30-40 см
5. Оставить запас у контроллера: 50-70 см
6. Сразу промаркировать кабель

**Типы кабелей:**
- КСПВ 4×0.5 — для обычных датчиков (сигнал + питание 12В)
- КСПВ 2×0.5 — для простых датчиков (только сигнал)
- UTP Cat5e — для IP-датчиков с PoE

**На что обратить внимание:**
- Не пережимать кабель
- Маркировка сразу после прокладки
- Отдельно от силовых 220В

**Типичные ошибки:**
- Нет маркировки — путаница при подключении
- Мало запаса — неудобно монтировать

**Время:** ~10-15 минут на линию

---

### Шаг 5: Подготовка точек для накладных датчиков

**Описание:** Для датчиков без скрытого монтажа (протечки, дым).

**Как делать:**
1. Вывести кабель в нужной точке
2. Закрепить конец кабеля
3. Для датчиков протечки — вывести на уровне пола
4. Для датчиков дыма — вывести к потолку
5. Защитить конец от повреждений
6. Промаркировать

**На что обратить внимание:**
- Длина выпуска: 30-50 см
- Кабель защищён от механических повреждений
- Место вывода соответствует проекту

**Типичные ошибки:**
- Кабель торчит — повреждение при отделке
- Нет запаса — неудобно подключать

**Время:** ~5-10 минут на точку

---

### Шаг 6: Проверка линий

**Описание:** Проверяем целостность всех кабелей.

**Как делать:**
1. Проверить мультиметром каждую жилу:
   - Сопротивление < 10 Ом (целая жила)
   - Между жилами — обрыв (нет КЗ)
2. Записать результаты проверки
3. Составить таблицу маркировки
4. Сфотографировать трассы

**На что обратить внимание:**
- Все жилы должны "прозваниваться"
- Нет короткого замыкания между жилами
- Документация для последующего монтажа

**Типичные ошибки:**
- Пропустить проверку — проблемы при настройке
- Нет документации — долго искать нужный кабель

**Время:** ~3-5 минут на линию

## Чек-лист качества

### Критические:

- [ ] Все линии прозваниваются — нет обрывов
- [ ] Нет короткого замыкания между жилами
- [ ] Кабели промаркированы
- [ ] Запас у датчиков достаточный (30-40 см)
- [ ] Подрозетники установлены ровно

### Важные:

- [ ] Трассы отдельно от силовых кабелей
- [ ] Гофра надёжно закреплена
- [ ] Есть фото трасс до закрытия
- [ ] Таблица маркировки составлена

### Желательные:

- [ ] Концы кабелей защищены
- [ ] Резервные линии проложены
- [ ] Документация передана заказчику

---

## Приёмка работы

### Что проверяет заказчик:

1. Количество точек соответствует проекту
2. Расположение точек правильное
3. Все линии работают (прозвонка)
4. Маркировка читаемая

### Допустимые отклонения:

- Расположение: ±50 мм
- Длина запаса: ±10 см

### Недопустимо:

- Неработающие линии
- Отсутствие маркировки
- Неправильное расположение
- Подрозетники выступают

## Техника безопасности

### Средства защиты (СИЗ):

- [ ] Очки защитные (при сверлении) — обязательно
- [ ] Респиратор (при сверлении бетона) — обязательно
- [ ] Перчатки — рекомендуется

### Опасности:

- Пыль при сверлении — респиратор
- Скрытая проводка — проверить детектором
- Работа на высоте — устойчивая стремянка

### Запрещено:

- Сверлить без проверки проводки
- Работать без СИЗ
- Использовать неисправный инструмент

## Частые вопросы (FAQ)

**Q: Какой кабель использовать для датчиков?**
A: КСПВ 4×0.5 — универсальный выбор (2 жилы питание, 2 жилы сигнал). Для простых датчиков — КСПВ 2×0.5. Для IP-датчиков — UTP Cat5e.

**Q: Нужны ли закладные для беспроводных датчиков?**
A: Нет, беспроводные датчики работают от батареек и не требуют прокладки кабелей. Но проводные надёжнее.

**Q: Какой запас кабеля оставлять?**
A: У датчика: 30-40 см. У контроллера: 50-70 см. Это позволит удобно подключить и при необходимости заменить датчик.

**Q: Можно ли прокладывать кабели датчиков вместе с интернетом?**
A: Да, слаботочные кабели можно прокладывать вместе. Главное — отдельно от силовых 220В.
