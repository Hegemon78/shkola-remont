# Демонтаж натяжного потолка

> Технология снятия натяжного потолка (плёночного или тканевого). Плёночный потолок можно снять с сохранением для повторной установки. Тканевый снимается сложнее.

---

## Когда применять

- Доступ к коммуникациям над потолком
- Замена потолка на новый
- Устранение последствий залива
- Ремонт основного потолка
- Установка дополнительных светильников

## Когда НЕ применять

- Для доступа к одной точке — можно сделать через отверстие под светильник
- Если потолок можно не трогать (работа у стен)

---

## Преимущества

- Быстрый демонтаж при знании технологии
- Плёночный потолок можно использовать повторно
- Минимум мусора
- Тихая работа

## Недостатки

- Нужен специальный инструмент (шпатели для натяжных)
- Плёнку легко порвать
- Тканевый потолок обычно одноразовый
- Профиль может повреждаться при снятии

---

## Необходимые материалы

| Материал | Характеристики | Количество | Примечание |
|----------|---------------|------------|------------|
| Перчатки | Тканевые, чистые | 1 пара | Не испачкать плёнку |

| Инструмент | Характеристики | Обязательно |

## Подготовка

### Проверить перед началом:

- [ ] Определить тип потолка (ПВХ-плёнка или ткань)
- [ ] Определить тип крепления (гарпунный, штапиковый, клипсовый)
- [ ] Отключить электричество (светильники)
- [ ] Освободить помещение от ценных вещей (тепловая пушка)

### Подготовить рабочее место:

1. Отключить электричество
2. Снять светильники из потолка
3. Убрать картины, цветы (от тепловой пушки)
4. Проветрить помещение

---

## Пошаговая инструкция

### Шаг 1: Снятие светильников и декоративной вставки

**Описание:** Подготавливаем потолок к демонтажу.

**Как делать:**
1. Отключить электричество
2. Снять точечные светильники (отжать пружины)
3. Снять люстру (если есть)
4. Вынуть декоративную вставку (маскировочную ленту)
5. Вставка просто вытягивается из паза профиля

**На что обратить внимание:**
- Вставка может быть приклеена — аккуратно поддеть
- Провода заизолировать
- Без вставки видно край плёнки/ткани

**Время:** ~15-20 минут

---

### Шаг 2: Демонтаж ПВХ-плёнки (гарпунная система)

**Описание:** Снимаем плёночный потолок с гарпунным креплением.

**Как делать:**
1. Нагреть плёнку тепловой пушкой до 50-60°C
2. Начать с угла помещения
3. Поддеть гарпун шпателем в углу
4. Потянуть гарпун плоскогубцами вниз и на себя
5. Продолжать по периметру, нагревая перед собой
6. Плёнка отстегнётся от профиля

**На что обратить внимание:**
- Плёнка без нагрева хрупкая — обязательно греть!
- Работать аккуратно — не порвать
- Гарпун — жёсткая кромка по краю плёнки
- Если планируется повторная установка — беречь плёнку

**Типичные ошибки:**
- Тянуть без нагрева — плёнка рвётся
- Царапать плёнку инструментом
- Перегреть — плёнка деформируется

**Время:** ~30-60 минут на комнату 20 м²

---

### Шаг 3: Демонтаж тканевого потолка (клипсовая система)

**Описание:** Снимаем тканевый потолок.

**Как делать:**
1. Нагрев НЕ нужен (ткань не эластичная)
2. Начать с середины стены (не с угла!)
3. Поддеть ткань шпателем
4. Вынуть край ткани из клипсы
5. Двигаться от центра к углам

**На что обратить внимание:**
- Ткань не растягивается как плёнка
- При повторной установке может не хватить материала
- Тканевые потолки обычно не используются повторно

**Время:** ~30-60 минут

---

### Шаг 4: Демонтаж штапиковой системы

**Описание:** Снимаем потолок со штапиковым креплением.

**Как делать:**
1. Нагреть плёнку (если ПВХ)
2. Вытащить штапик (клин) из профиля
3. Штапик — тонкая пластиковая рейка
4. Поддеть отвёрткой, вытянуть
5. После удаления штапика плёнка освободится

**На что обратить внимание:**
- Штапик часто повреждается при демонтаже
- Плёнка обрезается по размеру — при повторной установке может не хватить
- Нужен новый штапик для повторного монтажа

**Время:** ~30-60 минут

---

### Шаг 5: Демонтаж профиля (если нужен)

**Описание:** Снимаем крепёжный профиль со стен.

**Как делать:**
1. Выкрутить саморезы из профиля
2. Или сбить профиль (если не нужен)
3. Заделать отверстия в стене
4. Профиль можно использовать повторно

**Время:** ~30-60 минут

## Чек-лист качества

### Критические:

- [ ] Потолок полностью снят
- [ ] Плёнка не повреждена (если для повторного использования)
- [ ] Электричество безопасно отключено

### Важные:

- [ ] Профиль снят или готов к новому потолку
- [ ] Помещение проветрено

---

## Техника безопасности

### Средства защиты (СИЗ):

- [ ] Перчатки — обязательно (горячая плёнка)
- [ ] Очки — при работе над головой

### Опасности:

- **Тепловая пушка** — пожароопасно, не направлять на горючие предметы
- **Горячая плёнка** — можно обжечься
- **Электричество** — отключить светильники
- **Падение с высоты** — устойчивая стремянка

### Запрещено:

- Работать с тепловой пушкой рядом с горючими материалами
- Перегревать плёнку (температура выше 70°C)
- Работать без отключения электричества

## Частые вопросы (FAQ)

**Q: Можно ли использовать плёнку повторно?**
A: ПВХ с гарпунным креплением — да, если не повреждена. Штапиковое крепление — плёнка обрезана, может не хватить. Тканевый — обычно нет.

**Q: Как понять тип крепления?**
A: Гарпун — жёсткий край плёнки защёлкнут в профиль. Штапик — плёнка прижата клином. Клипса — ткань заправлена в пазы.

**Q: Обязательна ли тепловая пушка?**
A: Для ПВХ — да, иначе плёнка порвётся. Можно использовать мощный строительный фен. Для ткани — не нужна.

## Вопросы для теста

### Вопрос 1 (выбор одного):

Почему нужно нагревать ПВХ-плёнку перед демонтажем?

- [ ] Для дезинфекции
- [ ] Чтобы быстрее работать
- [x] Без нагрева плёнка хрупкая и рвётся
- [ ] Нагрев не нужен

**Объяснение:** ПВХ-плёнка при комнатной температуре жёсткая и хрупкая. При нагреве до 50-60°C она становится эластичной и легко снимается.

---

### Вопрос 2 (выбор одного):

С какого места начинать демонтаж тканевого потолка?

- [ ] С угла
- [x] С середины стены
- [ ] От светильника
- [ ] Не имеет значения

**Объяснение:** Тканевый потолок начинают снимать с середины стены, постепенно двигаясь к углам. Это предотвращает неравномерное натяжение и повреждение.
