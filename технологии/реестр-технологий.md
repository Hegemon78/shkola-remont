# Технологии

Детальные инструкции по выполнению работ.

---

## Что такое технология?

**Технология** — это конкретный способ выполнения процесса.

Один процесс может иметь несколько технологий:
- Процесс "Штробление" → Технологии: штроборезом, перфоратором
- Процесс "Покраска" → Технологии: валиком, краскопультом

---

## Структура технологии

Каждая технология содержит:

1. **Метаданные** — ID, связи, сложность
2. **Когда применять** — условия использования
3. **Материалы и инструменты** — что нужно
4. **Подготовка** — чек-лист перед началом
5. **Пошаговая инструкция** — шаги с фото
6. **Чек-лист качества** — критерии приёмки
7. **Техника безопасности** — СИЗ, опасности
8. **Вопросы для теста** — проверка знаний

| T-PK | Покраска |
