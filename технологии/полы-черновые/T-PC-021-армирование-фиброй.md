# Армирование фиброй

> Добавление фиброволокна в раствор стяжки для объёмного армирования. Альтернатива металлической сетке. Предотвращает усадочные трещины, повышает ударную прочность и износостойкость.

---

## Когда применять

- Стяжка на прочном основании (бетон)
- Полусухая стяжка (основной вид армирования)
- Дополнительное армирование к сетке
- Стяжка без тёплого пола
- Небольшие помещения (до 20 м²)
- Тонкие стяжки (30-40 мм)

## Когда НЕ применять

- Плавающая стяжка по утеплителю (нужна сетка)
- Стяжка с тёплым полом (рекомендуется сетка)
- Большие помещения без деформационных швов
- Высокие нагрузки (склады, гаражи) — только фибры недостаточно
- Когда требуется конструктивное армирование

---

## Преимущества

- Простота — добавляется в раствор при замесе
- Объёмное армирование — волокна во всём объёме стяжки
- Предотвращение усадочных трещин
- Повышение ударной прочности
- Не нужны фиксаторы и связывание
- Снижение расхода материала (нет отходов)
- Экономия времени

## Недостатки

- Не заменяет конструктивное армирование (сетку)
- Не работает при больших нагрузках
- Требует точной дозировки
- Может комковаться при неправильном замесе
- Волокна могут торчать из поверхности (нужна затирка)

---

## Подготовка

### Проверить перед началом:

- [ ] Тип фибры соответствует задаче
- [ ] Рассчитан расход фибры на объём стяжки
- [ ] Фибра сухая, не слипшаяся
- [ ] Есть пластификатор (рекомендуется)
- [ ] Миксер достаточной мощности

### Подготовить рабочее место:

1. Вскрыть упаковку с фиброй
2. Проверить — волокна распушены, не комками
3. Подготовить ёмкости для замеса
4. Отмерить порции фибры для каждого замеса

### Расчёт количества фибры:

**Формула:** Объём стяжки (м³) × Расход фибры (кг/м³)

**Пример:** Комната 20 м², стяжка 50 мм = 20 × 0.05 = 1 м³
Расход ПП фибры 0.9 кг/м³ → нужно 0.9 кг фибры

---

## Пошаговая инструкция

### Шаг 1: Расчёт дозировки на замес

**Описание:** Определение количества фибры на один замес раствора.

**Как делать:**
1. Определить объём одного замеса (обычно 40-60 литров)
2. Рассчитать количество фибры на замес
3. Пример: замес 50 л = 0.05 м³ × 0.9 кг/м³ = 45 грамм фибры
4. Отмерить порции заранее

**На что обратить внимание:**
- Точная дозировка важна
- Мало фибры — не работает
- Много фибры — ухудшается удобоукладываемость

**Типичные ошибки:**
- «На глазок» — неравномерное армирование
- Передозировка — раствор плохо укладывается

**Время:** ~5 минут (расчёт)

---

### Шаг 2: Подготовка сухих компонентов

**Описание:** Смешивание фибры с сухой смесью.

**Как делать (предпочтительный способ):**
1. Засыпать сухую смесь (пескобетон) в ёмкость
2. Добавить отмеренную порцию фибры
3. Перемешать всухую 1-2 минуты
4. Фибра должна распределиться равномерно

**Альтернативный способ:**
1. Засыпать сухую смесь в воду
2. Начать замес
3. Постепенно добавлять фибру в работающий миксер
4. Добавлять малыми порциями, не комками

**На что обратить внимание:**
- Фибра должна распределиться без комков
- При добавлении в воду — сыпать постепенно
- Увеличить время замеса на 1-2 минуты

**Типичные ошибки:**
- Высыпали всю фибру сразу — скомковалась
- Не перемешали всухую — неравномерное распределение

**Время:** ~2-3 минуты

---

### Шаг 3: Замес раствора

**Описание:** Приготовление раствора с фиброй.

**Как делать:**
1. Добавить воду (по инструкции к смеси)
2. Добавить пластификатор (рекомендуется)
3. Перемешивать миксером 3-5 минут
4. Оставить на 5 минут для созревания
5. Перемешать ещё 1-2 минуты

**На что обратить внимание:**
- Время замеса увеличивается из-за фибры
- Раствор должен быть однородным
- Волокна не должны торчать пучками

**Типичные ошибки:**
- Недостаточный замес — фибра комками
- Много воды — фибра всплывает наверх

**Время:** ~7-10 минут на замес

---

### Шаг 4: Проверка качества раствора

**Описание:** Контроль равномерности распределения фибры.

**Как делать:**
1. Набрать пробу раствора в руку
2. Осмотреть — волокна видны равномерно
3. Нет крупных комков фибры
4. Консистенция раствора нормальная

**На что обратить внимание:**
- Волокна торчат в разных направлениях
- Нет участков без волокон
- Раствор пластичный, укладывается нормально

**Типичные ошибки:**
- Не проверили — комки обнаружили при укладке

**Время:** ~1 минута

---

### Шаг 5: Укладка стяжки

**Описание:** Стандартная укладка стяжки (раствор уже армирован).

**Как делать:**
1. Укладывать раствор как обычно
2. Разравнивать правилом
3. При затирке — волокна уходят в толщу
4. Если волокна торчат — дополнительная затирка

**На что обратить внимание:**
- Раствор укладывается обычным способом
- При затирке волокна скрываются
- Финишная затирка — чистая поверхность

**Типичные ошибки:**
- Не затёрли — волокна торчат (эстетический дефект)
- Слишком сухой раствор — плохо укладывается

**Время:** Стандартное для стяжки

## Приёмка работы

### Что проверяет заказчик:

1. Однородность стяжки — нет расслоений
2. Отсутствие трещин (после высыхания)
3. Поверхность — волокна не торчат
4. Документация — сертификат на фибру

### Допустимые отклонения:

- Единичные волокна на поверхности — допустимо (удаляются при шлифовке)
- Отклонение дозировки ±10%

### Недопустимо:

- Комки фибры в стяжке
- Участки без фибры
- Большое количество волокон на поверхности
- Использование неподходящего типа фибры

---

## Техника безопасности

### Средства защиты (СИЗ):

- [ ] Перчатки — рекомендуется (стекловолокно колется)
- [ ] Респиратор — при работе со стекло/базальтовой фиброй
- [ ] Очки — при пылении

### Опасности:

- Стекловолокно — раздражение кожи и дыхательных путей
- Базальтовая фибра — может раздражать кожу
- Полипропиленовая — наиболее безопасная

### Запрещено:

- Работать со стекловолокном без СИЗ
- Вдыхать пыль при засыпке фибры
- Использовать фибру неизвестного происхождения

## Частые вопросы (FAQ)

**Q: Может ли фибра полностью заменить сетку?**
A: При стяжке на прочном основании — да. При плавающей стяжке по утеплителю или тёплом поле — нет, нужна сетка. Фибра предотвращает усадочные трещины, но не обеспечивает конструктивное армирование.

**Q: Какой тип фибры лучше?**
A: Полипропиленовая — универсальная, дешёвая, безопасная. Базальтовая — прочнее, но дороже. Стальная — для промышленных полов. Для квартиры достаточно ПП фибры.

**Q: Сколько фибры добавлять?**
A: Полипропиленовая: 0.6-0.9 кг на 1 м³ раствора (примерно 600-900 г на комнату 20 м² при стяжке 50 мм).

**Q: Можно ли добавить больше фибры для надёжности?**
A: Не рекомендуется. Избыток фибры ухудшает удобоукладываемость раствора и может вызвать расслоение. Следуйте рекомендациям производителя.

**Q: Почему волокна торчат из стяжки?**
A: Недостаточная затирка. При финишной затирке волокна уходят в толщу. Оставшиеся на поверхности — срезаются или шлифуются.
