# Установка приточного клапана

> Технология монтажа пассивного приточного клапана в наружную стену для естественного притока свежего воздуха. Простое решение для вентиляции помещений.

---

## Когда применять

- Пассивная приточная вентиляция
- Квартиры с герметичными окнами
- Дополнение к вытяжной вентиляции
- Бюджетное решение для притока воздуха

## Когда НЕ применять

- Нужен подогрев воздуха — выбрать бризер
- Очень шумная улица — нужен клапан с шумоглушением
- Холодный климат без отопления под клапаном

---

## Необходимые материалы

| Материал | Характеристики | Количество | Примечание |
|----------|---------------|------------|------------|
| Приточный клапан | KIV, Домвент, Вентс | 1 шт | По производительности |
| Утеплитель | Минвата или ППЭ | По длине трубы | Для теплоизоляции |
| Герметик | Силиконовый | 1 туба | Для уплотнения |
| Монтажная пена | Для фасада | 1 баллон | Заделка зазоров |

## Необходимые инструменты

| Инструмент | Характеристики | Обязательно |
|------------|---------------|-------------|
| Установка алмазного бурения | Ø100-160 мм | Да |
| Перфоратор | Для крепления | Да |
| Уровень | 40-60 см | Да |
| Шуруповёрт | С насадками | Да |

---

## Пошаговая инструкция

### Шаг 1: Выбор места установки

**Как делать:**
1. Выбрать место на наружной стене
2. Оптимально — над радиатором отопления (подогрев воздуха)
3. Высота 2-2.2 м от пола
4. Проверить отсутствие коммуникаций в стене

**На что обратить внимание:**
- Над радиатором — воздух будет подогреваться
- Не напротив кровати
- Не в углу (плохая циркуляция)

**Время:** ~10 минут

---

### Шаг 2: Сверление отверстия

**Как делать:**
1. Разметить центр отверстия
2. Диаметр по размеру клапана (обычно Ø100-132 мм)
3. Сверлить с уклоном 3-5° вниз наружу
4. Удалить керн, очистить отверстие

**Время:** ~30-45 минут

---

### Шаг 3: Установка трубы

**Как делать:**
1. Вставить трубу воздуховода в отверстие
2. Обрезать по толщине стены + 10 мм
3. Обернуть утеплителем
4. Зафиксировать, загерметизировать пеной

**Время:** ~15-20 минут

---

### Шаг 4: Монтаж клапана

**Как делать:**
1. Установить наружную решётку
2. Вставить фильтр и шумоглушитель
3. Закрепить внутреннюю часть клапана
4. Проверить работу заслонки

**Время:** ~15-20 минут

---

## Чек-лист качества

- [ ] Отверстие с уклоном наружу
- [ ] Труба утеплена
- [ ] Зазоры загерметизированы
- [ ] Заслонка открывается/закрывается
- [ ] Нет сквозняка при закрытом клапане
- [ ] Наружная решётка установлена

---

## Техника безопасности

- [ ] Защитные очки при сверлении
- [ ] Респиратор от пыли
- [ ] Страховка при работе снаружи
