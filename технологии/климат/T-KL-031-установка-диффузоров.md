# Установка диффузоров

> Технология монтажа приточных и вытяжных диффузоров с возможностью регулировки воздушного потока. Обеспечивают равномерное распределение воздуха.

---

## Когда применять

- Приточно-вытяжная вентиляция с разводкой
- Канальные кондиционеры
- Требуется регулировка потока воздуха
- Требуется равномерное распределение

## Когда НЕ применять

- Простая вытяжка — достаточно решётки
- Не нужна регулировка

---

## Типы диффузоров

| Тип | Применение | Особенности |
|-----|------------|-------------|
| Круглый приточный | Подача воздуха | Веерное распределение |
| Круглый вытяжной | Удаление воздуха | Сбор воздуха |
| Квадратный | Подвесные потолки | 600×600, 300×300 мм |
| Щелевой | Вдоль стен, окон | Линейное распределение |
| Сопловый | Большие помещения | Дальнобойный поток |

## Пошаговая инструкция

### Шаг 1: Подготовка места установки

**Как делать:**
1. Найти выход воздуховода
2. Снять защитную заглушку
3. Очистить край отверстия
4. Проверить диаметр — должен соответствовать диффузору

**Время:** ~5-10 минут

---

### Шаг 2: Установка адаптера (если нужен)

**Как делать:**
1. Вставить адаптер в воздуховод
2. Закрепить саморезами или защёлками
3. Загерметизировать соединение
4. Проверить надёжность

**Время:** ~10-15 минут

---

### Шаг 3: Установка диффузора

**Как делать:**
1. Вставить диффузор в адаптер
2. Выровнять по уровню (для квадратных)
3. Закрепить защёлками или саморезами
4. Проверить фиксацию

**На что обратить внимание:**
- Диффузор должен плотно прилегать к потолку
- Не должно быть щелей
- Центральная часть должна быть съёмной (для обслуживания)

**Время:** ~10-15 минут

---

### Шаг 4: Настройка воздушного потока

**Как делать:**
1. Открыть регулятор на максимум
2. Проверить поток воздуха
3. При необходимости отрегулировать направление
4. Настроить интенсивность потока

**Время:** ~5-10 минут

## Чек-лист качества

- [ ] Диффузор установлен ровно

- [ ] Регулятор потока работает
- [ ] Воздух распределяется равномерно
- [ ] Нет посторонних шумов
