# Установка внутреннего блока канального типа

> Технология монтажа канального внутреннего блока кондиционера за подвесным потолком с разводкой воздуховодов по помещениям.

---

## Когда применять

- Скрытый кондиционер за потолком
- Кондиционирование нескольких зон
- Требования к дизайну (блок не виден)
- Высота потолков позволяет (от 2.7 м)

## Когда НЕ применять

- Низкие потолки — нет места для блока
- Одно небольшое помещение — достаточно настенного
- Нет подвесного потолка

---

## Необходимые материалы

| Материал | Характеристики | Количество | Примечание |
|----------|---------------|------------|------------|
| Канальный блок | По мощности | 1 шт | Низко/средненапорный |
| Воздуховоды гибкие | Ø150-250 мм | По проекту | Для разводки |
| Шпильки М8 | Длина по высоте | 4 шт | Для подвеса блока |
| Виброизоляторы | Резиновые | 4 шт | Снижение шума |
| Решётки/диффузоры | По количеству точек | По проекту | Приток и рециркуляция |

## Необходимые инструменты

| Инструмент | Характеристики | Обязательно |
|------------|---------------|-------------|
| Перфоратор | С бурами | Да |
| Уровень | 60-100 см | Да |
| Труборез | Для меди | Да |
| Развальцовка | Метрическая/дюймовая | Да |
| Динамометрический ключ | Для гаек | Да |
| Ножницы по металлу | Для воздуховодов | Да |

---

## Пошаговая инструкция

### Шаг 1: Разметка и подготовка

**Как делать:**
1. Определить место установки блока за потолком
2. Учесть доступ для обслуживания (ревизионный люк)
3. Разметить точки подвеса
4. Разметить трассы воздуховодов

**На что обратить внимание:**
- Доступ к фильтрам для обслуживания
- Место для дренажного насоса (если нужен)
- Расстояние до приточных решёток

**Время:** ~20-30 минут

---

### Шаг 2: Установка подвесов

**Как делать:**
1. Просверлить отверстия в потолке
2. Установить анкеры
3. Вкрутить шпильки
4. Установить виброизоляторы
5. Выровнять по высоте

**Время:** ~30-40 минут

---

### Шаг 3: Монтаж блока

**Как делать:**
1. Поднять блок (работать вдвоём)
2. Закрепить на подвесах через виброизоляторы
3. Выровнять горизонтально
4. Затянуть крепления

**На что обратить внимание:**
- Блок должен быть строго горизонтален
- Виброизоляция обязательна
- Учесть уклон для дренажа

**Время:** ~30-40 минут

---

### Шаг 4: Подключение фреоновых труб

**Как делать:**
1. Подготовить трубы (обрезать, снять фаску, развальцевать)
2. Подключить к блоку
3. Затянуть динамометрическим ключом
4. Изолировать соединения

**Время:** ~20-30 минут

---

### Шаг 5: Монтаж воздуховодов

**Как делать:**
1. Подключить приточный воздуховод к блоку
2. Развести воздуховоды к точкам раздачи
3. Подключить рециркуляционный воздуховод (забор воздуха)
4. Закрепить, загерметизировать стыки
5. Утеплить воздуховоды

**Время:** ~1-2 часа

---

### Шаг 6: Подключение дренажа

**Как делать:**
1. Подключить дренажную трубку
2. При необходимости — установить дренажный насос
3. Вывести дренаж в канализацию или наружу
4. Проверить уклон

**Время:** ~20-30 минут

---

## Чек-лист качества

- [ ] Блок горизонтален
- [ ] Виброизоляция установлена
- [ ] Фреоновые трубы подключены правильно
- [ ] Воздуховоды подключены и загерметизированы
- [ ] Дренаж работает
- [ ] Есть доступ для обслуживания

---

## Техника безопасности

- [ ] Работать вдвоём при подъёме блока
- [ ] Устойчивые подмости
- [ ] Не включать до вакуумирования
