# Монтаж рекуператора

> Технология установки приточно-вытяжной установки с рекуперацией тепла. Обеспечивает приток свежего воздуха с подогревом за счёт тепла вытяжного воздуха.

---

## Когда применять

- Энергоэффективная вентиляция
- Холодный климат
- Требуется приток и вытяжка одновременно
- Экономия на отоплении

## Когда НЕ применять

- Достаточно простого приточного клапана
- Нет места для двух отверстий (приток и вытяжка)
- Бюджетное решение — выбрать бризер

## Пошаговая инструкция

### Шаг 1: Выбор места и разметка

**Как делать:**
1. Выбрать место на наружной стене
2. Учесть расстояние между отверстиями (по модели)
3. Высота 2-2.2 м от пола
4. Проверить отсутствие коммуникаций
5. Разметить оба отверстия

**Время:** ~15-20 минут

---

### Шаг 2: Сверление отверстий

**Как делать:**
1. Просверлить отверстие для притока
2. Просверлить отверстие для вытяжки
3. Оба с уклоном 3-5° наружу
4. Очистить отверстия

**На что обратить внимание:**
- Расстояние между отверстиями по инструкции
- Не перепутать приток и вытяжку

**Время:** ~45-60 минут

---

### Шаг 3: Установка воздуховодов

**Как делать:**
1. Вставить воздуховоды в отверстия
2. Утеплить по всей длине
3. Загерметизировать зазоры
4. Установить наружные решётки

**Время:** ~30-40 минут

---

### Шаг 4: Монтаж рекуператора

**Как делать:**
1. Закрепить монтажную пластину
2. Установить рекуператор
3. Подключить воздуховоды
4. Подключить электропитание
5. Установить крышку

**Время:** ~30-40 минут

---

### Шаг 5: Пусконаладка

**Как делать:**
1. Включить рекуператор
2. Проверить направление потоков
3. Настроить скорость
4. Проверить работу рекуперации
5. Объяснить управление

**Время:** ~15-20 минут

---

## Чек-лист качества

- [ ] Оба отверстия с уклоном наружу
- [ ] Воздуховоды утеплены
- [ ] Приток и вытяжка не перепутаны
- [ ] Рекуператор работает на всех режимах
- [ ] Нет конденсата внутри
- [ ] Наружные решётки установлены

---

## Техника безопасности

- [ ] Защитные очки при сверлении
- [ ] Респиратор от пыли
- [ ] Отключение электричества при подключении
