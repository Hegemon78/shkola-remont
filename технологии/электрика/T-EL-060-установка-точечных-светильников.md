# Установка точечных светильников

> Монтаж встраиваемых точечных светильников в подвесные потолки (ГКЛ, натяжные, реечные). Финишный этап после монтажа потолка.

---

## Когда применять

- Подвесной потолок из ГКЛ
- Натяжной потолок
- Реечный потолок
- Потолок Армстронг

## Когда НЕ применять

- Бетонный потолок без подшивки — накладные светильники
- Нет разводки освещения

---

## Преимущества

- Равномерное освещение
- Скрытый монтаж
- Зонирование светом
- Современный вид

## Недостатки

- Требуется подвесной потолок
- Ограничение по мощности (нагрев)
- Нужен доступ к проводке

---

## Необходимые материалы

| Материал | Характеристики | Расход | Примечание |
|----------|---------------|--------|------------|
| Точечные светильники | GU5.3, GU10 или GX53 | По проекту | LED предпочтительно |
| Лампы | LED, соответствующий цоколь | По количеству | 3000K или 4000K |
| Клеммники WAGO | 2-3 контакта | По количеству | Для подключения |
| Термокольца | Для натяжного потолка | По количеству | Защита от нагрева |

## Необходимые инструменты

| Инструмент | Характеристики | Обязательно |
|------------|---------------|-------------|
| Коронка по ГКЛ | Ø по размеру светильника | Да (для ГКЛ) |
| Дрель/шуруповёрт | С регулировкой оборотов | Да |
| Стриппер | Для зачистки | Да |
| Индикаторная отвёртка | Для проверки | Да |
| Отвёртка | Плоская (для пружин) | Да |

## Подготовка

### Проверить перед началом:

- [ ] Потолок смонтирован
- [ ] Кабель выведен в точках
- [ ] Известны размеры светильников (диаметр)
- [ ] ЭЛЕКТРИЧЕСТВО ОТКЛЮЧЕНО

### Подготовить рабочее место:

1. Отключить автомат освещения
2. Проверить отсутствие напряжения
3. Найти выводы кабелей

## Пошаговая инструкция

### Шаг 1: Разметка отверстий

**Описание:** Размечаем точки установки светильников.

**Как делать:**
1. По проекту отметить центры светильников
2. Проверить равномерность расстояний
3. Учесть расположение каркаса (не попасть на профиль)
4. Для ГКЛ — проверить магнитом положение профилей

**На что обратить внимание:**
- Расстояние между светильниками (обычно 1-1.5 м)
- Отступ от стены (обычно 50-70 см)
- Не попасть на профиль каркаса

**Типичные ошибки:**
- Попасть на профиль — не вырезать отверстие
- Неравномерное расстояние

**Время:** ~10-15 минут

---

### Шаг 2: Вырезание отверстий (для ГКЛ)

**Описание:** Высверливаем отверстия под светильники.

**Как делать:**
1. Определить диаметр посадочного отверстия
2. Выбрать коронку нужного диаметра
3. Сверлить без удара, на средних оборотах
4. Придерживать коронку, не давить

**На что обратить внимание:**
- Диаметр указан на корпусе светильника
- Типовые размеры: 55, 68, 75, 85, 105 мм
- Аккуратные края — виден ободок

**Типичные ошибки:**
- Неправильный диаметр — светильник не встанет или провалится
- Рваные края

**Время:** ~1-2 минуты на отверстие

---

### Шаг 3: Для натяжного потолка

**Описание:** Установка светильников в натяжной потолок.

**Как делать:**
1. Закладные установлены ДО натяжки
2. Найти закладные через полотно
3. Наклеить термокольцо по центру
4. Вырезать полотно внутри кольца
5. Вытянуть провод

**На что обратить внимание:**
- Термокольцо обязательно — защита от нагрева
- Резать только внутри кольца
- Не повредить полотно

**Типичные ошибки:**
- Без термокольца — полотно оплавится
- Порез за пределами кольца

**Время:** ~3-5 минут на точку

---

### Шаг 4: Вытягивание провода

**Описание:** Достаём провод из отверстия.

**Как делать:**
1. Найти провод в полости потолка
2. Вытянуть через отверстие
3. Оставить 15-20 см запаса
4. Если провод далеко — использовать крючок

**На что обратить внимание:**
- Провод должен быть заложен заранее
- Достаточный запас для подключения

**Типичные ошибки:**
- Провод не заложен — протягивать сложно
- Короткий провод

**Время:** ~1-2 минуты

---

### Шаг 5: Подключение светильника

**Описание:** Подключаем светильник к проводке.

**Как делать:**
1. Зачистить провода (10 мм)
2. Подключить через клеммник WAGO:
   - Фаза (коричневый) к L
   - Ноль (синий) к N
   - Земля (жёлто-зелёный) к PE (если есть)
3. Проверить надёжность соединения

**На что обратить внимание:**
- Патрон может быть несъёмным — сначала подключить
- Земля, если есть в светильнике

**Типичные ошибки:**
- Забыть подключить землю

**Время:** ~2-3 минуты

---

### Шаг 6: Установка светильника

**Описание:** Устанавливаем светильник в отверстие.

**Как делать:**
1. Сжать пружины (ушки)
2. Завести светильник в отверстие
3. Отпустить пружины — они зафиксируют
4. Выровнять, если есть поворотный механизм

**На что обратить внимание:**
- Пружины должны защёлкнуться за потолок
- Светильник сидит плотно
- Провода не пережаты

**Типичные ошибки:**
- Пружины не защёлкнулись — светильник выпадет
- Пережат провод

**Время:** ~1 минута

---

### Шаг 7: Установка лампы

**Описание:** Устанавливаем лампу в светильник.

**Как делать:**
1. Вставить лампу в патрон:
   - GU5.3/GU10 — вставить и повернуть
   - GX53 — вставить и повернуть до щелчка
2. Для MR16 — установить стопорное кольцо

**На что обратить внимание:**
- Не трогать галогенные лампы руками (жир — перегрев)
- LED можно трогать
- Лампа зафиксирована

**Типичные ошибки:**
- Лампа не зафиксирована — выпадет

**Время:** ~30 секунд

---

### Шаг 8: Проверка работы

**Описание:** Проверяем работу освещения.

**Как делать:**
1. Включить автомат
2. Включить выключатель
3. Проверить работу всех светильников
4. Проверить направление (если поворотные)

**На что обратить внимание:**
- Все светильники горят
- Нет мерцания
- Цветовая температура одинаковая

**Типичные ошибки:**
- Не проверить — обнаружится при сдаче

**Время:** ~2-3 минуты

## Частые вопросы (FAQ)

**Q: LED или галогенки?**
A: LED — экономичнее, меньше нагрев, дольше служат. Для натяжных — только LED.

**Q: Какая мощность для комнаты?**
A: Для LED ~5 Вт на 1 м² (т.е. для 15 м² нужно ~75 Вт или 10 ламп по 7 Вт).

**Q: Какую цветовую температуру выбрать?**
A: 3000K — тёплый (спальня, гостиная), 4000K — нейтральный (кухня, ванная), 6000K — холодный (офис).

**Q: Как заменить лампу?**
A: Вытащить стопорное кольцо, достать лампу, заменить, вставить кольцо обратно.
