# Монтаж распределительных коробок

> Установка распределительных (распаечных) коробок для соединения кабелей и разветвления электропроводки. Ключевой узел электрической сети квартиры.

---

## Когда применять

- Разветвление линий на группы
- Соединение кабелей освещения
- Подключение выключателей
- Переход между комнатами
- Узлы скрытой проводки

## Когда НЕ применять

- Распайка в подрозетниках — для простых схем
- Шлейфовое подключение розеток — коробка не нужна

---

## Преимущества

- Доступ к соединениям
- Удобство обслуживания
- Возможность модификации сети
- Организованное соединение кабелей

## Недостатки

- Дополнительные элементы
- Нужен доступ после отделки
- Требуется качественное соединение

---

## Необходимые материалы

| Материал | Характеристики | Расход | Примечание |
|----------|---------------|--------|------------|
| Распредкоробка | Ø 80-100 мм, IP20/IP55 | По проекту | Для скрытой/открытой |
| Клеммники WAGO | 2-5 контактов | По количеству соединений | Или СИЗ |
| СИЗ (колпачки) | Разных размеров | По количеству | Альтернатива WAGO |
| Гильзы ГМЛ | Для опрессовки | По количеству | Надёжное соединение |
| Алебастр | Для фиксации | ~50 г на коробку | Для скрытой установки |

## Необходимые инструменты

| Инструмент | Характеристики | Обязательно |
|------------|---------------|-------------|
| Перфоратор | С коронкой Ø 80-100 мм | Да (скрытая) |
| Шуруповёрт | Для накладных | Да |
| Стриппер | Для снятия изоляции | Да |
| Пресс-клещи | Для гильз ГМЛ | При опрессовке |
| Отвёртка | Для клемм | Да |
| Мультиметр | Для проверки | Да |

### Проверить перед началом:

- [ ] Есть схема распайки

- [ ] Известно расположение коробок по проекту
- [ ] Электричество отключено

### Подготовить рабочее место:

1. Обесточить линии
2. Подготовить инструмент и материалы
3. Разметить положение коробок

---

## Пошаговая инструкция

### Шаг 1: Выбор места установки

**Описание:** Определяем оптимальное положение коробки.

**Как делать:**
1. Коробки над выключателями (15-20 см от потолка)
2. Или в углу комнаты
3. Должен быть доступ после отделки
4. Не за мебелью

**На что обратить внимание:**
- Доступ для обслуживания
- Минимальная длина кабелей
- Эстетика (можно скрыть за карнизом)

**Типичные ошибки:**
- Коробка за шкафом — нет доступа
- Слишком далеко — длинные кабели

**Время:** ~5 минут на комнату

---

### Шаг 2: Подготовка отверстия (скрытая установка)

**Описание:** Высверливаем лунку для коробки.

**Как делать:**
1. Разметить центр
2. Высверлить коронкой Ø 80-100 мм
3. Выбить центр перфоратором
4. Очистить от пыли
5. Смочить водой или загрунтовать

**На что обратить внимание:**
- Глубина = высота коробки + 5 мм
- Лунка без острых выступов

**Типичные ошибки:**
- Мелкая лунка — коробка выпирает
- Не очистить — плохая адгезия

**Время:** ~5-10 минут на коробку

---

### Шаг 3: Заведение кабелей

**Описание:** Вводим кабели в коробку.

**Как делать:**
1. Выломать заглушки в нужных местах
2. Завести кабели в коробку
3. Оставить запас 10-15 см внутри
4. Зафиксировать кабели (сальники или хомуты)

**На что обратить внимание:**
- Достаточный запас для соединения
- Кабели не должны вытягиваться
- Герметичные вводы для влажных помещений

**Типичные ошибки:**
- Короткий запас — неудобно соединять
- Кабели болтаются — могут выдернуться

**Время:** ~3-5 минут на коробку

---

### Шаг 4: Установка коробки

**Описание:** Фиксируем коробку в стене или на поверхности.

**Как делать:**
1. Для скрытой: замесить алебастр
2. Нанести в лунку
3. Вставить коробку
4. Выровнять заподлицо
5. Для накладной: просверлить, закрепить дюбелями

**На что обратить внимание:**
- Коробка не должна выпирать
- Крышка должна прилегать к стене
- Для ванной — IP55

**Типичные ошибки:**
- Коробка выпирает — крышка не закроется
- Перекос — некрасиво

**Время:** ~5-10 минут на коробку

---

### Шаг 5: Разделка кабелей

**Описание:** Готовим кабели к соединению.

**Как делать:**
1. Снять внешнюю оболочку кабеля (10-12 см)
2. Снять изоляцию с жил (10-12 мм для WAGO, 25-30 мм для СИЗ)
3. Разложить жилы по группам: L, N, PE
4. Маркировать если нужно

**На что обратить внимание:**
- Не повредить изоляцию жил
- Для WAGO — зачистка строго по метке
- Жилы не должны касаться друг друга

**Типичные ошибки:**
- Повреждение изоляции — КЗ
- Слишком короткая зачистка

**Время:** ~3-5 минут на кабель

---

### Шаг 6: Соединение жил

**Описание:** Соединяем жилы по схеме.

**Как делать:**
1. **WAGO:** вставить жилы до упора в клеммник
2. **СИЗ:** скрутить жилы, накрутить колпачок
3. **Опрессовка:** вставить в гильзу, обжать прессом
4. Соединять по группам: фаза с фазой, ноль с нолём, земля с землёй

**На что обратить внимание:**
- WAGO только для одножильных (или спец. серия)
- СИЗ подбирать по сечению и количеству жил
- Опрессовка — самое надёжное соединение

**Типичные ошибки:**
- Неправильное соединение по схеме
- Слабый контакт — нагрев
- Смешивание фазы и нуля

**Время:** ~5-10 минут на коробку

---

### Шаг 7: Укладка и закрытие

**Описание:** Аккуратно укладываем соединения.

**Как делать:**
1. Уложить соединения в коробку
2. Не переламывать жилы
3. Группы отдельно друг от друга
4. Закрыть крышку

**На что обратить внимание:**
- Соединения не должны касаться крышки
- Жилы не пережаты
- Крышка защёлкивается

**Типичные ошибки:**
- Соединения мешают закрыть крышку
- Жилы переломлены при укладке

**Время:** ~2-3 минуты на коробку

---

### Шаг 8: Проверка

**Описание:** Проверяем правильность соединений.

**Как делать:**
1. Прозвонить линии мультиметром
2. Проверить отсутствие КЗ
3. Проверить соответствие схеме
4. Подать напряжение и проверить работу

**На что обратить внимание:**
- Проверять ДО подачи напряжения
- Проверить все группы

**Типичные ошибки:**
- Не проверить — КЗ при включении

**Время:** ~5-10 минут на коробку

## Чек-лист качества

### Критические:

- [ ] Соединения по схеме верны
- [ ] Нет КЗ (прозвонено)
- [ ] Контакты надёжные (не болтаются)
- [ ] Коробка доступна для обслуживания

### Важные:

- [ ] Кабели зафиксированы
- [ ] Запас жил достаточный
- [ ] Крышка закрывается
- [ ] IP соответствует помещению

### Желательные:

- [ ] Аккуратная укладка
- [ ] Маркировка групп

## Приёмка работы

### Что проверяет заказчик:

1. Работоспособность освещения
2. Доступность коробок
3. Качество соединений (визуально)

### Допустимые отклонения:

- Положение: ± 20 мм

### Недопустимо:

- Скрутки без изоляции
- Нагрев соединений
- Недоступные коробки

## Техника безопасности

### Средства защиты (СИЗ):

- [ ] Перчатки диэлектрические (при работе под напряжением)
- [ ] Очки (при сверлении)

### Опасности:

- Удар током — обесточить линии
- Пыль — очки при сверлении

### Запрещено:

- Работать под напряжением
- Соединять скруткой без изоляции
- Соединять медь с алюминием напрямую

## Частые вопросы (FAQ)

**Q: WAGO или СИЗ?**
A: WAGO удобнее и быстрее, но дороже. СИЗ надёжны при правильной установке. Опрессовка — для ответственных соединений.

**Q: Можно ли без коробки?**
A: При шлейфовом подключении розеток и распайке в подрозетниках — можно.

**Q: Какой IP для ванной?**
A: Минимум IP44, лучше IP55.

**Q: Как соединить медь и алюминий?**
A: Только через клеммник или специальные гильзы. Напрямую нельзя — окисление.
