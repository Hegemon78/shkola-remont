# Сборка телескопической коробки

> Сборка дверной коробки телескопического типа с пазами для наличников и доборов. Современный вариант, позволяющий регулировать глубину и устанавливать наличники без гвоздей.

---

## Когда применять

- Коробка телескопического типа (с пазами)
- Планируется установка телескопических наличников
- Требуется возможность регулировки глубины
- Толщина стены требует доборов

## Когда НЕ применять

- Коробка стандартного типа (без пазов) — T-DI-011
- Бюджетный вариант без телескопической системы
- Коробка уже собрана производителем

---

## Преимущества

- Регулировка глубины посадки
- Наличники без гвоздей (в паз)
- Скрытое крепление
- Возможность замены наличников

## Недостатки

- Дороже стандартной коробки
- Требует точной сборки
- Сложнее в регулировке

---

## Подготовка

### Проверить перед началом:

- [ ] Комплектность коробки (2 стойки, перекладина)
- [ ] Отсутствие повреждений на элементах
- [ ] Размеры соответствуют полотну
- [ ] Определено направление открывания двери

### Подготовить рабочее место:

1. Расчистить ровную площадку на полу
2. Защитить пол от повреждений (картон, плёнка)
3. Подготовить инструменты
4. Достать полотно для примерки

---

## Пошаговая инструкция

### Шаг 1: Раскладка элементов

**Описание:** Раскладываем элементы коробки и определяем их положение.

**Как делать:**
1. Разложить все элементы на полу
2. Определить петлевую стойку (со стороны петель)
3. Определить притворную стойку (со стороны замка)
4. Положить перекладину сверху
5. Сориентировать пазы наружу (для наличников)

**На что обратить внимание:**
- Пазы должны быть направлены в сторону стены
- Притвор (выступ) направлен внутрь коробки
- Все элементы должны быть одного цвета/текстуры

**Типичные ошибки:**
- Перепутать стороны — притвор не совпадёт
- Неправильное направление пазов — наличники не встанут

**Время:** ~5 минут

---

### Шаг 2: Определение размеров

**Описание:** Рассчитываем размеры элементов коробки.

**Как делать:**
1. Измерить высоту полотна
2. Добавить зазор снизу (10-15 мм от пола)
3. Добавить верхний зазор (3 мм)
4. Толщина перекладины (обычно 30 мм)
5. Итого высота стойки = полотно + 10-15 + 3 + 30 мм
6. Ширина перекладины = полотно + 2×3 мм (зазоры) + 2×30 мм (стойки)

**На что обратить внимание:**
- Стандартная высота стойки ~2070-2080 мм
- Учитывать толщину чистового пола
- Проверить высоту проёма

**Типичные ошибки:**
- Не учесть зазоры — дверь будет цеплять
- Забыть про пол — дверь упрётся

**Время:** ~5 минут

---

### Шаг 3: Разметка запила

**Описание:** Размечаем линии запила под 45° или 90°.

**Как делать:**
1. Отмерить нужную длину стойки
2. Разметить линию запила под 45° (классика) или 90° (современный)
3. Аналогично разметить вторую стойку
4. Разметить перекладину с двух сторон
5. Проверить — стойки должны быть одинаковой длины

**На что обратить внимание:**
- Запил 45° — углы коробки под углом (классический вид)
- Запил 90° — прямые углы (современный вид)
- Направление запила важно — не перепутать

**Типичные ошибки:**
- Разная длина стоек — перекос коробки
- Неправильное направление угла — не сойдётся

**Время:** ~10 минут

---

### Шаг 4: Запил элементов

**Описание:** Выполняем точный запил торцов.

**Как делать:**
1. Установить торцовочную пилу на 45° (или 90°)
2. Надёжно зафиксировать заготовку
3. Выполнить рез плавно, без рывков
4. Проверить качество реза — без сколов
5. Повторить для всех элементов

**На что обратить внимание:**
- Диск должен быть чистым и острым
- Резать со стороны лицевой поверхности
- Не допускать сколов МДФ

**Типичные ошибки:**
- Тупой диск — сколы на поверхности
- Резкое движение — неровный рез
- Неточный угол — щели на стыках

**Время:** ~15 минут

---

### Шаг 5: Засверливание отверстий

**Описание:** Сверлим отверстия под саморезы для предотвращения раскола.

**Как делать:**
1. Сложить стойку и перекладину встык
2. Разметить 2 точки для саморезов
3. Засверлить отверстия сверлом 2.5-3 мм
4. Сверлить через перекладину в стойку
5. Повторить для второго угла

**На что обратить внимание:**
- Отверстия на расстоянии 15-20 мм от краёв
- Сверло тоньше самореза
- Не просверлить насквозь лицевую сторону

**Типичные ошибки:**
- Без засверливания — МДФ раскалывается
- Слишком близко к краю — скол

**Время:** ~10 минут

---

### Шаг 6: Сборка углов

**Описание:** Соединяем элементы коробки саморезами.

**Как делать:**
1. Зафиксировать соединение струбцинами
2. Нанести клей ПВА на торцы (опционально)
3. Вкрутить саморезы — не перетягивать
4. Проверить угол угольником (90° в углах коробки)
5. Повторить для второго угла

**На что обратить внимание:**
- Саморезы не должны выходить на лицевую сторону
- Угол между стойкой и перекладиной = 90°
- Соединение должно быть плотным

**Типичные ошибки:**
- Перетянуть саморезы — треснет МДФ
- Не проверить угол — перекос коробки
- Выход саморезов — испорчен внешний вид

**Время:** ~15 минут

---

### Шаг 7: Проверка собранной коробки

**Описание:** Контролируем качество сборки.

**Как делать:**
1. Положить коробку на ровный пол
2. Проверить прямоугольность — измерить диагонали
3. Приложить полотно — проверить зазоры
4. Зазор 3 мм по бокам и сверху
5. Проверить отсутствие щелей на стыках

**На что обратить внимание:**
- Диагонали должны быть равны (±1-2 мм)
- Полотно должно свободно входить
- Стыки без видимых щелей

**Типичные ошибки:**
- Не проверить диагонали — коробка перекошена
- Слишком маленький зазор — дверь будет цеплять

**Время:** ~10 минут

## Чек-лист качества

> Проверить после завершения работы

### Критические (нельзя сдать без выполнения):

- [ ] Коробка собрана прямоугольно (диагонали равны)
- [ ] Углы запилены точно (без щелей на стыках)
- [ ] Полотно свободно входит в коробку
- [ ] Зазоры 3 мм по периметру

### Важные:

- [ ] Саморезы не видны с лицевой стороны
- [ ] МДФ не расколот
- [ ] Пазы для наличников не повреждены

### Желательные:

- [ ] Стыки проклеены для усиления
- [ ] Коробка без царапин и повреждений

---

## Приёмка работы

### Что проверяет заказчик:

1. Коробка собрана без видимых дефектов
2. Стыки аккуратные, без щелей
3. Полотно входит с правильными зазорами

### Допустимые отклонения:

- Диагонали: разница ≤ 2 мм
- Щель на стыке: ≤ 0.5 мм
- Зазор полотно-коробка: 3 ± 1 мм

### Недопустимо:

- Видимые сколы на лицевой стороне
- Трещины МДФ
- Щели на углах > 1 мм
- Саморезы вышли наружу

## Техника безопасности

### Средства защиты (СИЗ):

- [ ] Защитные очки (при пилении) — обязательно
- [ ] Наушники (при работе с пилой) — рекомендуется
- [ ] Перчатки — при необходимости

### Опасности:

- Порезы пильным диском — не держать руки близко к диску
- Отлетающие частицы — защитные очки
- Шум — наушники при длительной работе

### Запрещено:

- Работать без защитного кожуха пилы
- Держать заготовку руками при резе
- Отвлекаться во время работы с пилой

## Частые вопросы (FAQ)

**Q: Чем телескопическая коробка отличается от стандартной?**
A: Телескопическая имеет пазы для установки наличников и доборов без гвоздей. Стандартная — гладкая, наличники крепятся на клей или гвозди.

**Q: Можно ли обрезать телескопическую коробку?**
A: Да, но аккуратно, чтобы не повредить пазы. Рез должен быть чистым.

**Q: Какой клей использовать для усиления соединений?**
A: Столярный ПВА или специальный клей для МДФ. Не использовать суперклей — он хрупкий.
