# Установка глазка / видеоглазка

> Технология установки оптического или электронного глазка на входную дверь для просмотра посетителей.

---

## Когда применять

- Дверь без глазка
- Замена старого глазка
- Установка видеоглазка с записью
- Требуется широкий угол обзора

## Когда НЕ применять

- Есть видеодомофон — глазок не нужен
- Дверь со стеклянной вставкой

---

## Типы глазков

### Оптический (стандартный)
- Угол обзора 120-200°
- Не требует питания
- Простая установка

### Видеоглазок
- Камера + экран внутри
- Запись посетителей
- Ночное видение
- Требует батареек/питания

---

## Преимущества

- Возможность видеть посетителя
- Простая установка
- Недорого (оптический)

## Недостатки

- Оптический — нужно подходить к двери
- Видеоглазок — требует питания
- Заслонка снаружи — можно закрыть

---

## Необходимые материалы

| Материал | Характеристики | Расход | Примечание |
|----------|---------------|--------|------------|
| Глазок | По толщине двери (60-100 мм) | 1 шт | Угол обзора 180° |
| Батарейки | AA или аккумулятор | 2-4 шт | Для видеоглазка |

## Необходимые инструменты

| Инструмент | Характеристики | Обязательно |
|------------|---------------|-------------|
| Дрель | | Да |
| Коронка по металлу | 12-16 мм (под глазок) | Да |
| Рулетка | | Да |
| Маркер | | Да |
| Отвёртка | Плоская (для разборки глазка) | Да |

## Пошаговая инструкция

### Шаг 1: Определение места установки

**Как делать:**
1. Определить высоту (на уровне глаз — 150-160 см от пола)
2. Центрировать по ширине двери
3. Убедиться, что внутри нет рёбер жёсткости
4. Отметить точку сверления с обеих сторон

**На что обратить внимание:**
- Учесть рост всех жильцов
- Для видеоглазка может потребоваться большее отверстие

**Время:** ~10 минут

---

### Шаг 2: Сверление отверстия

**Как делать:**
1. Наклеить малярный скотч в месте сверления (защита покрытия)
2. Начать сверление с внутренней стороны
3. Не просверливать насквозь сразу!
4. Когда сверло выйдет наружу — сверлить с другой стороны навстречу
5. Это предотвратит сколы и задиры

**На что обратить внимание:**
- Диаметр отверстия = диаметру глазка (обычно 12-16 мм)
- Работать на низких оборотах с металлом

**Время:** ~15 минут

---

### Шаг 3: Обработка отверстия

**Как делать:**
1. Снять заусенцы напильником
2. Обработать края антикоррозийным составом
3. Примерить глазок

**Время:** ~10 минут

---

### Шаг 4: Установка глазка

**Оптический глазок:**
1. Разобрать глазок (внешняя и внутренняя части)
2. Вставить внешнюю часть (объектив) снаружи
3. Накрутить внутреннюю часть изнутри
4. Затянуть вручную или плоской отвёрткой (не перетягивать!)

**Видеоглазок:**
1. Вставить камеру снаружи
2. Провести кабель через отверстие
3. Закрепить внутренний модуль (экран)
4. Подключить питание
5. Настроить (дата, время, режимы)

**Время:** ~15-20 минут

---

### Шаг 5: Проверка

**Как делать:**
1. Проверить обзор — должно быть видно всю площадку
2. Проверить чёткость изображения
3. Для видеоглазка — проверить запись, ночной режим
4. Убедиться, что глазок не шатается

**Время:** ~5 минут

---

## Чек-лист качества

### Критические:

- [ ] Глазок установлен надёжно, не шатается
- [ ] Изображение чёткое
- [ ] Виден весь обзор (площадка, посетитель)

### Важные:

- [ ] Глазок на удобной высоте
- [ ] Нет повреждений покрытия двери
- [ ] Видеоглазок: работает запись

---

## Выбор глазка

| Параметр | Рекомендация |
|----------|--------------|
| Угол обзора | 180° и более |
| Длина корпуса | По толщине двери + 5 мм |
| Диаметр | 12-16 мм (стандарт) |
| Материал | Латунь, сталь (не пластик) |

---

## Техника безопасности

### СИЗ:

- [ ] Защитные очки при сверлении
- [ ] Перчатки (острые края)
