# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Обзор проекта

«Виртуальная Школа Ремонта» — система стандартизации и обучения строительным/ремонтным работам. Цель: обучить рабочих → протестировать → сертифицировать → нанять на реальные объекты.

## Иерархия сущностей

```
НАПРАВЛЕНИЕ (Direction) — специализация (Электрика, Сантехника, Отделка...)
    └── ЭТАП (Stage) — Черновая / Чистовая
            └── ПРОЦЕСС (Process) — конкретная работа (Штробление, Покраска...)
                    └── ТЕХНОЛОГИЯ (Technology) — способ выполнения (валиком, штроборезом...)
                            └── ШАГ (Step) — пошаговая инструкция
                                    └── ЧЕК-ПОИНТ (Checkpoint) — критерий качества
```

## Структура проекта

```
/направления/          — 19 направлений работ (D00-D18)
    реестр-направлений.md  — сводный список всех направлений
    /электрика/

/этапы/                — логика последовательности (черновые-работы.md, чистовые-работы.md)

    реестр-технологий.md   — сводный список всех технологий

```

## ID-система

**Направления:** D00–D18 (D00 = Культура производства, D02 = Электрика, D04 = Сантехника...)

**Технологии:** `T-[направление]-[номер]`
- T-EL = Электрика, T-SN = Сантехника, T-PK = Покраска, T-PL = Плитка и т.д.
- Пример: `T-EL-001` — первая технология электрики

## Именование файлов

- **Направления:** `D02-электрика.md` (ID + название)
- **Материалы:** `мат-электрика.md` (префикс + направление)
- **Реестры:** `реестр-технологий.md`, `реестр-направлений.md`
- **Технологии:** `T-EL-001-штробление-штроборезом.md` (ID + краткое название)
