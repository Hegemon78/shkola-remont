#### 1. Определена концепция проекта
- **Цель:** Стандартизация процессов → Обучение рабочих → Сертификация → Найм
- **Целевая аудитория:** Рабочие для найма на реальные объекты
- **Формат:** Текст + фото → тестирование → очная приёмка
- **Геймификация:** Значки, чекпоинты, шкала прогресса
- **Платформа:** Веб / Telegram, вход частично бесплатный

- Иерархия: Направление → Этап → Процесс → Технология → Шаг → Чек-поинт

#### 4. Проработана логика этапов

**Черновые работы (10 этапов):**
1. Культура производства (постоянно)
2. Демонтаж
3. Возведение стен и перегородок
4. Разводка коммуникаций (электрика, слаботочка, сантехника, климат)
5. Коммуникации в полу
6. Стяжка пола (армирование, маяки, заливка)
7. Прокладка проводов + обшивка ГКЛ
8. Штукатурка
9. Подготовка стен (шпаклёвка, армировка)
10. Окна / Входная дверь

**Чистовые работы (13 этапов):**
1. Плитка (пол → стены → фартук)
2. Чистовые полы → укрывка
3. Подоконники
4. Покраска / Обои
5. Потолочный декор
6. Натяжные потолки
7. Карнизы для штор
8. Электрика финишная
9. Сантехника финишная
10. Климат финишная
11. Двери межкомнатные
12. Плинтусы, порожки
13. Финальная уборка

Файлы: `этапы/черновые-работы.md`, `этапы/чистовые-работы.md`

#### 5. Создана структура 19 направлений

Каждое направление в отдельной папке с файлом направления (D[XX]-название.md):
- Описание направления
- Таблица процессов
- Требования к рабочему
- Материалы и инструменты
- Связанные направления

#### 6. Ключевые решения

- **Логика связей:** Направление → Процессы → Технологии → Шаги → Чек-листы
- Процессы описываются отдельно в каждом направлении (даже если пересекаются)
- Контент в markdown файлах (ручное редактирование)
- Рабочий может выбрать несколько направлений
- Гофра обязательна для всех проводов
- Опрессовка — обязательный этап сантехники
- Укрывочные работы — на каждом этапе

#### 7. Подключён GitHub

- Репозиторий: https://github.com/Hegemon78/shkola-remont
- Версионирование всех изменений
