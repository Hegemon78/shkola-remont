# Материалы для подготовки стен

> Справочник материалов для грунтования, шпаклевания, армирования и шлифовки поверхностей.

---

## Грунтовки

### Типы грунтовок

| Тип | Применение | Особенности |
|-----|-----------|-------------|
| Глубокого проникновения | Пористые минеральные основания | Укрепляет, снижает впитываемость |
| Бетоноконтакт | Гладкие невпитывающие основания | Создаёт шероховатость, содержит кварц |
| Укрепляющая | Рыхлые, осыпающиеся основания | Глубокое проникновение, укрепление |
| Универсальная | Любые минеральные основания | Компромиссный вариант |
