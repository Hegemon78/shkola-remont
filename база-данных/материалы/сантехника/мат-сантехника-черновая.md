# Сантехника — Черновые материалы

Материалы для скрытых сантехнических работ: водоснабжение, канализация, отопление.

---

## Трубы водоснабжения

### Типы труб

| Тип | Применение | Плюсы | Минусы |
|-----|------------|-------|--------|
| **ППР (полипропилен)** | Стандарт для квартир | Дёшево, надёжно | Нужен паяльник, не гнётся |
| **Металлопластик** | Альтернатива ППР | Гнётся, быстрый монтаж | Дороже, фитинги слабое место |
| **Сшитый полиэтилен (PEX)** | Тёплый пол, скрытый монтаж | Гибкий, надёжный | Дорого, нужен инструмент |
| **Медь** | Премиум | Вечная, антибактериальная | Очень дорого |
| **Нержавейка (гофра)** | Подводка, сложные места | Гибкая, быстрый монтаж | Дорого для магистралей |

---

### ППР — Полипропилен

| Тип | Маркировка | Применение | Max температура |
|-----|------------|------------|-----------------|
| PN10 | Тонкостенная | Холодная вода | 20°C |
| PN20 | Стандарт | ХВС + ГВС | 60°C |
| PN25 | Армированная | ГВС, отопление | 95°C |

**Диаметры:**
- 20 мм — разводка по квартире
- 25 мм — стояки, магистрали
- 32 мм — ввод, коллекторы

**Нюансы:**
- PN25 — только для ГВС и отопления
- Армированная стекловолокном — не нужно зачищать
- Армированная алюминием — нужна зачистка перед пайкой

**Бренды:** см. [бренды/сантехника.md](../../бренды/сантехника.md#трубы)

---

### Фитинги ППР

| Тип | Применение |
|-----|------------|
| Муфта | Соединение труб |
| Угол 90° / 45° | Повороты |
| Тройник | Разветвление |
| Переход | Смена диаметра |
| Комбинированный (с резьбой) | Подключение к металлу |
| Кран шаровый | Отключение участка |
| Американка | Разборное соединение |

---

## Канализация

### Трубы

| Диаметр | Применение |
|---------|------------|
| 32 мм | Умывальник, биде |
| 40 мм | Мойка, душ (слив) |
| 50 мм | Ванна, душ, стиральная машина |
| 110 мм | Унитаз, стояк |

**Материал:**
- Серый ПВХ — внутренняя канализация
- Оранжевый ПВХ — наружная (НЕ использовать внутри)

**Нюансы:**
- Уклон: 2-3 см на 1 метр
- Раструбы — против потока воды
- Ревизии — на каждом этаже и поворотах

---

### Фитинги канализации

| Тип | Применение |
|-----|------------|
| Отвод 45° / 90° | Повороты |
| Тройник | Подключение приборов |
| Крестовина | Соединение 4 труб |
| Ревизия | Прочистка |
| Переход | Смена диаметра |
| Компенсатор | Температурное расширение |

---

## Отопление (черновая)

### Трубы для отопления

| Тип | Применение |
|-----|------------|
| ППР PN25 | Радиаторное отопление |
| Сшитый полиэтилен PEX | Тёплый пол |
| Металлопластик | Тёплый пол, радиаторы |
| Медь | Премиум, долговечность |

---

### Тёплый пол

| Элемент | Назначение |
|---------|------------|
| Труба PEX 16/20 мм | Контуры тёплого пола |
| Коллектор | Распределение по контурам |
| Насосно-смесительный узел | Регулировка температуры |
| Демпферная лента | Компенсация расширения |
| Теплоизоляция (ЭППС) | Подложка под трубы |
| Сетка / маты | Крепление труб |

**Нюансы:**
- Шаг укладки: 10-20 см
- Длина контура: до 80-100 м
- Давление опрессовки: 6 бар

---

## Запорная арматура

| Тип | Применение |
|-----|------------|
| Шаровый кран | Полное перекрытие |
| Вентиль | Регулировка потока |
| Обратный клапан | Защита от обратного потока |
| Фильтр грубой очистки | Перед счётчиками |
| Редуктор давления | Стабилизация давления |
| Гибкая подводка | Подключение приборов |

**Нюансы:**
- Краны только полнопроходные (не шароверные)
- Фильтр — обязательно перед счётчиками
- Редуктор — если давление > 4 бар

---

## Гидроизоляция (сантехузлы)

| Материал | Применение |
|----------|------------|
| Обмазочная (мастика) | Стандарт для ванных |
| Рулонная | Полы с большой нагрузкой |
| Лента гидроизоляционная | Углы, примыкания |
| Манжеты | Проходы труб |

## Чек-лист закупки (черновая сантехника)

### Водоснабжение:
- [ ] Трубы ППР (PN20 ХВС, PN25 ГВС)
- [ ] Фитинги ППР по схеме
- [ ] Краны шаровые на каждый прибор
- [ ] Фильтры грубой очистки
- [ ] Счётчики воды
- [ ] Обратные клапаны
- [ ] Коллекторы (если лучевая разводка)

- [ ] Фитинги по схеме
