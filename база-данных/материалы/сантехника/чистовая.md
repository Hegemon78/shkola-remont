# Сантехника — Чистовые материалы

Материалы для финишного этапа: сантехприборы, смесители, аксессуары.

---

## Смесители

### Типы по конструкции

| Тип | Особенности | Применение |
|-----|-------------|------------|
| Однорычажный | Стандарт, удобный | Везде |
| Двухвентильный | Классика, ретро-стиль | Интерьерное решение |
| Термостатический | Поддержание температуры | Душ, ванна |
| Сенсорный | Бесконтактный | Общественные места |
| Настенный | Скрытый монтаж | Современный дизайн |

### Типы по месту установки

| Место | Тип смесителя |
|-------|---------------|
| Раковина | Однорычажный, высокий/низкий излив |
| Кухня | С выдвижным изливом, с фильтром |
| Ванна | С душем, каскадный |
| Душ | Встраиваемый, термостат |
| Биде | Специальный с аэратором |

**Критерии выбора:**
- Картридж: керамика (не пластик)
- Размер картриджа: 35-40 мм (надёжнее)
- Аэратор: съёмный для чистки
- Гибкая подводка: нержавейка в оплётке

**Нюансы:**
- Дешёвые смесители = частая замена картриджа
- Для жёсткой воды — с фильтром или отдельный фильтр
- Термостат требует давления > 2 бар

**Бренды:** см. [бренды/сантехника.md](../../бренды/сантехника.md#смесители)

## Раковины

| Тип | Особенности |
|-----|-------------|
| Накладная | На столешницу, современный стиль |
| Врезная | В столешницу сверху |
| Подстольная | Под столешницу, удобно протирать |
| Подвесная | На стену, без тумбы |
| На пьедестале | Классика, скрывает коммуникации |
| Угловая | Для маленьких помещений |

**Материалы:**
- Фаянс/фарфор — стандарт
- Искусственный камень — премиум, тяжёлый
- Нержавейка — кухня, подсобка
- Стекло — дизайнерские решения

---

## Унитазы

### Типы

| Тип | Особенности |
|-----|-------------|
| Напольный | Классика, простой монтаж |
| Подвесной | Современный, легко мыть пол |
| Приставной | Компромисс, бачок скрыт |
| Компакт | Бачок на унитазе |

### Выпуск

| Тип выпуска | Применение |
|-------------|------------|
| Горизонтальный | Стандарт для новостроек |
| Косой | Старый фонд, хрущёвки |
| Вертикальный | Частные дома, редко |

### Смыв

| Тип | Особенности |
|-----|-------------|
| Круговой (душевой) | Лучше омывает |
| Прямой (каскадный) | Проще, дешевле |

**Нюансы:**
- Инсталляция для подвесного = отдельная покупка
- Антивсплеск — геометрия чаши
- Микролифт — плавное опускание крышки

---

## Ванны

| Материал | Плюсы | Минусы |
|----------|-------|--------|
| Акрил | Лёгкая, тёплая, разные формы | Царапается |
| Сталь | Дешёвая, прочная | Шумная, быстро остывает |
| Чугун | Вечная, тихая, тёплая | Очень тяжёлая |
| Искусственный камень | Премиум, любые формы | Дорого, тяжёлая |

**Размеры:**
- Стандарт: 150×70, 170×70
- Увеличенные: 170×75, 180×80
- Сидячие: 120×70

---

## Душевые

### Душевые кабины

| Тип | Особенности |
|-----|-------------|
| С поддоном | Простой монтаж |
| Без поддона (walk-in) | Современно, нужен трап |
| Угловая | Экономия места |
| Пристенная | Больше пространства |

### Душевые системы

| Элемент | Варианты |
|---------|----------|
| Лейка | Ручная, верхняя (тропический), боковая |
| Штанга | Простая, с полкой |
| Стойка | С термостатом, с изливом |
| Шланг | 150-200 см, металл/пластик |

---

## Полотенцесушители

| Тип | Особенности |
|-----|-------------|
| Водяной | Подключение к ГВС/отоплению |
| Электрический | Независимый, с терморегулятором |
| Комбинированный | Оба режима |

**Материалы:**
- Нержавейка — долговечная
- Латунь с покрытием — дизайнерские
- Чёрная сталь — бюджет (красить)
