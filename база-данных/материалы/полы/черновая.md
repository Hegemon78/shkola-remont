# Полы — Черновые материалы

Материалы для подготовки оснований: стяжки, выравнивание, тёплый пол.

---

## Стяжки

### Типы стяжек

| Тип | Толщина | Применение |
|-----|---------|------------|
| Цементно-песчаная (ЦПС) | 40-100 мм | Классика, прочная |
| Полусухая | 40-80 мм | Быстрее сохнет |
| Наливной пол | 3-30 мм | Финишное выравнивание |
| Сухая (ГВЛ) | 20-100 мм | Без мокрых процессов |

### Цементно-песчаная стяжка

**Пропорции:** 1:3 (цемент М400 : песок)

**Расход на 1 м² при толщине 50 мм:**
- Цемент: ~18 кг
- Песок: ~50 кг

**Армирование:**
- Сетка 100×100×4 мм при толщине > 40 мм
- Фибра — альтернатива сетке

**Нюансы:**
- Минимальная толщина: 40 мм
- На утеплитель: минимум 50 мм с армированием
- Созревание: 28 дней (ходить через 3-5 дней)

---

### Готовые смеси для стяжки

| Тип | Применение | Плюсы |
|-----|------------|-------|
| Пескобетон М300 | Стяжки, основания | Готовая смесь |
| Стяжка для тёплого пола | Под обогрев | Эластичная |
| Быстротвердеющая | Сжатые сроки | Ходить через 4-6 часов |

---

### Наливные полы

| Тип | Толщина | Применение |
|-----|---------|------------|
| Базовый | 10-80 мм | Грубое выравнивание |
| Универсальный | 3-30 мм | Средние неровности |
| Финишный | 1-10 мм | Идеальная поверхность |

**Расход:** ~1.5-1.8 кг/м² на 1 мм толщины

**Нюансы:**
- Грунтовать обязательно
- Прокатывать игольчатым валиком
- Не допускать сквозняков

**Бренды:** см. [бренды/отделка.md](../../бренды/отделка.md#наливные-полы)

## Сухая стяжка

### Компоненты

| Элемент | Назначение |
|---------|------------|
| Керамзит фр. 0-5 мм | Засыпка |
| Элементы пола ГВЛ | Настил |
| Кромочная лента | По периметру |
| Полиэтилен плёнка | Пароизоляция |
| Саморезы для ГВЛ | Крепление |
| Клей ПВА | Склейка замков |

**Плюсы:**
- Нет мокрых процессов
- Можно укладывать покрытие сразу
- Хорошая звукоизоляция

**Минусы:**
- Боится воды
- Меньшая несущая способность

## Тёплый пол

### Водяной тёплый пол

| Элемент | Варианты |
|---------|----------|
| Труба | PEX-A, PEX-B, PE-RT, металлопластик |
| Диаметр | 16 мм (стандарт), 20 мм (большие контуры) |
| Теплоизоляция | ЭППС 30-50 мм, профильные маты |
| Демпферная лента | По периметру помещения |
| Коллектор | На количество контуров + запас |
| Смесительный узел | Регулировка температуры |

**Параметры:**
- Шаг укладки: 10-20 см
- Длина контура: до 80-100 м
- Толщина стяжки над трубой: минимум 30 мм

---

### Электрический тёплый пол

| Тип | Применение | Под покрытие |
|-----|------------|--------------|
| Кабель в стяжку | Основной обогрев | Любое |
| Маты нагревательные | Подогрев | Плитка |
| Инфракрасная плёнка | Подогрев | Ламинат, паркет |
| Стержневой | Под плитку | Плитка |

**Мощность:**
- Комфортный подогрев: 100-150 Вт/м²
- Основной обогрев: 150-200 Вт/м²

---

## Гидроизоляция пола

| Тип | Применение |
|-----|------------|
| Обмазочная (мастика) | Ванные, кухни |
| Рулонная наплавляемая | Перекрытия, подвалы |
| Рулонная самоклеящаяся | Простой монтаж |
| Проникающая | Бетонные основания |

**Правила:**
- Заводить на стены 10-20 см
- Армировать углы лентой
- 2 слоя минимум

---

## Звукоизоляция пола

| Материал | Толщина | Снижение шума |
|----------|---------|---------------|
| Вспененный полиэтилен | 2-5 мм | Ударный шум |
| Пробка | 2-6 мм | Ударный шум |
| Минвата под стяжку | 20-50 мм | Воздушный + ударный |
| Звукоизоляционные маты | 3-20 мм | Ударный шум |

---

## Утепление пола

| Материал | Толщина | Применение |
|----------|---------|------------|
| ЭППС | 30-100 мм | Первые этажи, под ТП |
| Минвата жёсткая | 50-100 мм | Под стяжку |
| Керамзит | 100-200 мм | Черновое утепление |
