# Отделка — Черновые материалы

Материалы для подготовки поверхностей: штукатурка, шпаклёвка, грунтовка.

---

## Штукатурные смеси

### Типы штукатурок

| Тип | Применение | Слой |
|-----|------------|------|
| Гипсовая | Внутренние работы, сухие помещения | до 50 мм |
| Цементная | Влажные помещения, фасады | до 30 мм |
| Цементно-известковая | Универсальная | до 30 мм |
| Машинная гипсовая | Большие объёмы | до 50 мм |

**Расход:**
- Гипсовая: ~8-9 кг/м² при слое 10 мм
- Цементная: ~15-16 кг/м² при слое 10 мм

**Нюансы:**
- Гипс на гипс — ОК, цемент на гипс — ❌
- Максимальный слой за один проход — см. инструкцию
- Армирование сеткой при слое > 30 мм

**Бренды:** см. [бренды/отделка.md](../../бренды/отделка.md#штукатурки)

## Шпаклёвки

### Типы

| Тип | Применение | Слой |
|-----|------------|------|
| Стартовая (базовая) | Выравнивание после штукатурки | 1-5 мм |
| Финишная | Подготовка под покраску/обои | 0.5-2 мм |
| Универсальная | Старт + финиш | 1-3 мм |
| Полимерная | Идеальная поверхность | до 3 мм |

**Расход:**
- Стартовая: ~1-1.2 кг/м² при слое 1 мм
- Финишная: ~0.5-0.7 кг/м² при слое 0.5 мм

**Нюансы:**
- Под обои достаточно 1-2 слоя финишной
- Под покраску — 2-3 слоя + шлифовка P180-P240
- Полимерная — для идеальных поверхностей (потолки)

---

## Грунтовки

| Тип | Применение |
|-----|------------|
| Глубокого проникновения | Рыхлые, пористые основания |
| Универсальная | Стандартные поверхности |
| Бетоноконтакт | Гладкий бетон перед штукатуркой |
| Адгезионная | Между разными материалами |
| Антигрибковая | Влажные помещения |

**Расход:** 100-200 мл/м²

**Правило:** Грунтовать между КАЖДЫМ слоем!

## Армирующие материалы

|----------|------------|

## Гипсокартон и комплектующие

### Типы ГКЛ

| Тип | Маркировка | Применение |
|-----|------------|------------|
| Обычный | ГКЛ (серый) | Сухие помещения |
| Влагостойкий | ГКЛВ (зелёный) | Ванные, кухни |
| Огнестойкий | ГКЛО (розовый) | Противопожарные требования |
| Влаго-огнестойкий | ГКЛВО | Специальные условия |

**Толщина:**
- 9.5 мм — потолки
- 12.5 мм — стены (стандарт)
- 2×12.5 мм — усиленные перегородки

### Профили

| Тип | Применение |
|-----|------------|
| ПН (направляющий) | По периметру |
| ПС (стоечный) | Вертикальные стойки |
| ПП (потолочный) | Потолочные конструкции |
| ППН | Направляющий для потолка |
| UA (усиленный) | Дверные проёмы |

### Комплектующие

| Элемент | Назначение |
|---------|------------|
| Подвес прямой | Крепление к основанию |
| Соединитель "краб" | Крестообразное соединение |
| Удлинитель профиля | Наращивание длины |
| Саморезы TN 25/35 | Крепление ГКЛ к профилю |
| Саморезы LN 9 ("клопы") | Соединение профилей |
| Дюбель-гвоздь | Крепление к стене/потолку |
| Лента уплотнительная | Звукоизоляция по периметру |
