# Климат — Чистовые материалы

Оборудование для финишного монтажа: кондиционеры, вентиляция, увлажнители.

---

## Кондиционеры (сплит-системы)

### Типы внутренних блоков

| Тип | Применение | Особенности |
|-----|------------|-------------|
| Настенный | Жильё, офисы | Стандарт, легко монтировать |
| Кассетный | Офисы, торговля | Потолочный монтаж |
| Канальный | Скрытый монтаж | Воздуховоды в потолке |
| Напольно-потолочный | Универсальный | 2 варианта установки |
| Колонный | Большие помещения | Высокая мощность |

### Выбор мощности

| Площадь | Мощность охлаждения |
|---------|---------------------|
| 15-20 м² | 2.0-2.5 кВт (07) |
| 20-25 м² | 2.5-3.0 кВт (09) |
| 25-35 м² | 3.0-3.5 кВт (12) |
| 35-50 м² | 5.0-5.5 кВт (18) |
| 50-70 м² | 7.0 кВт (24) |

**Поправки:**
- +20% если южная сторона
- +20% если много техники
- +10% на каждого человека сверх 2-х

### Инверторные vs On/Off

| Параметр | Инвертор | On/Off |
|----------|----------|--------|
| Энергопотребление | Меньше 30-40% | Больше |
| Шум | Тише | Громче |
| Точность температуры | ±0.5°C | ±2-3°C |
| Цена | Дороже | Дешевле |
| Срок службы | Дольше | Короче |

**Рекомендация:** только инвертор для жилья

**Бренды:** см. [бренды/климат.md](../../бренды/климат.md#кондиционеры)

## Мульти-сплит системы

Один внешний блок — несколько внутренних.

**Плюсы:**
- Один внешний блок на фасаде
- Экономия места

**Минусы:**
- Дороже нескольких сплитов
- При поломке внешнего — всё встаёт
- Длинные трассы — дороже монтаж

## Вентиляционное оборудование

### Бытовые рекуператоры

| Тип | Производительность | Особенности |
|-----|-------------------|-------------|
| Приточный клапан | 30-50 м³/ч | Пассивный |
| Бризер | 100-200 м³/ч | Приток с подогревом |
| Рекуператор настенный | 100-150 м³/ч | Приток + вытяжка |
| ПВУ центральная | 300-1000 м³/ч | На всю квартиру |

### Вытяжные вентиляторы

| Место | Производительность | Особенности |
|-------|-------------------|-------------|
| Ванная | 80-100 м³/ч | Влагозащита IP44 |
| Туалет | 50-80 м³/ч | С таймером |
| Кухня | 150-250 м³/ч | Или вытяжка |

**Подбор:** объём помещения × кратность воздухообмена
- Ванная: ×8-10
- Туалет: ×8-10
- Кухня: ×10-12

---

### Решётки и диффузоры

| Тип | Применение |
|-----|------------|
| Решётка вытяжная | Вытяжка в санузлах |
| Решётка приточная | Приток в комнаты |
| Диффузор круглый | Равномерная подача |
| Диффузор щелевой | Скрытый монтаж |
| Переточная решётка | В дверях/стенах |

## Увлажнители / Осушители

### Увлажнители

| Тип | Особенности |
|-----|-------------|
| Традиционный | Естественное испарение |
| Ультразвуковой | Мелкодисперсный туман |
| Паровой | Кипячение воды |
| Мойка воздуха | Увлажнение + очистка |

**Производительность:** 300-500 мл/ч для комнаты 20-30 м²

### Осушители

| Производительность | Применение |
|--------------------|------------|
| 10-20 л/сутки | Квартиры |
| 30-50 л/сутки | Большие помещения |
