# Материалы для оконных работ

> Справочник материалов для установки окон, подоконников, отливов и отделки откосов.

---

## Оконные конструкции

### Типы оконных профилей

| Тип | Плюсы | Минусы | Применение |
|-----|-------|--------|------------|
| ПВХ (пластик) | Доступная цена, не требует ухода, хорошая теплоизоляция | Ширина профиля, не экологично (для некоторых) | 90% жилых помещений |
| Дерево | Экологичность, эстетика, долговечность | Требует ухода, дороже, горючий | Частные дома, премиум |
| Алюминий | Узкий профиль, современный вид, долговечность | Холодный (нужен термомост), дорогой | Фасадное остекление, офисы |
| Дерево-алюминий | Дерево внутри + алюминий снаружи | Очень дорого | Премиум сегмент |

### Классы ПВХ профилей

| Класс | Толщина стенок | Особенности |
|-------|---------------|-------------|
| А | Наружные ≥ 2.8 мм, внутренние ≥ 2.5 мм | Стандарт для жилья |
| B | Наружные ≥ 2.5 мм, внутренние ≥ 2.0 мм | Бюджетный |
| C | Без нормирования | Только для нежилых помещений |

### Количество камер в профиле

| Камеры | Ширина профиля | Теплоизоляция | Применение |
|--------|----------------|---------------|------------|
| 3 | 58-62 мм | Средняя | Южные регионы |
| 5 | 70-76 мм | Хорошая | Средняя полоса |
| 6-7 | 80-90 мм | Высокая | Северные регионы |

### Рекомендуемые бренды профилей

| Бренд | Страна | Класс | Особенности |
|-------|--------|-------|-------------|
| REHAU | Германия | Премиум | Эталон качества, широкая линейка |
| KBE | Германия | Премиум/Бизнес | Экологичный, без свинца |
| VEKA | Германия | Премиум | Высокое качество, популярен в Европе |
| Schüco | Германия | Премиум | Алюминий и ПВХ, архитектурные решения |
| Salamander | Германия | Бизнес | Хорошее соотношение цена/качество |
| Proplex | Россия | Эконом | Бюджетный вариант |
| Novotex | Россия | Эконом | Доступный |

## Стеклопакеты

### Типы стеклопакетов

| Тип | Формула | Теплоизоляция | Применение |
|-----|---------|---------------|------------|
| Однокамерный | 4-16-4 | Низкая (R=0.32) | Балконы, нежилые |
| Двухкамерный | 4-10-4-10-4 | Средняя (R=0.47) | Стандарт для жилья |
| С энергосбережением | 4-16-4И | Высокая (R=0.59) | Северные регионы |
| Мультифункциональный | 4MF-16Ar-4 | Очень высокая | Энергоэффективные дома |

### Виды стёкол

| Тип | Особенности | Применение |
|-----|-------------|------------|
| Обычное флоат | Стандартное | Массовое применение |
| И-стекло (Low-E) | Напыление, отражает тепло внутрь | Энергосбережение |
| Мультифункциональное | Отражает тепло летом и зимой | Южные и северные фасады |
| Тонированное | Защита от солнца | Южные окна |
| Закалённое | Повышенная прочность | Безопасность |
| Триплекс | Не разлетается при разбитии | Нижние этажи, безопасность |

### Заполнение камер

| Газ | Теплопроводность | Цена |
|-----|------------------|------|
| Воздух | 1 (базовая) | - |
| Аргон | 0.67 (улучшение ~25%) | +500-1000 ₽/м² |
| Криптон | 0.36 (улучшение ~50%) | Очень дорого |

## Ленты и герметизация

### Типы лент

| Лента | Назначение | Где применять |
|-------|-----------|---------------|
| ПСУЛ | Паропроницаемая, гидрозащита | Снаружи, по ГОСТ |
| Пароизоляционная | Защита пены изнутри | Изнутри, по ГОСТ |
| Гидроизоляционная | Защита пены снаружи | Снаружи (альтернатива ПСУЛ) |
| Диффузионная | Паропроницаемая | Снаружи |

### ПСУЛ (предварительно сжатая уплотнительная лента)

| Параметр | Значение |
|----------|----------|
| Степень сжатия | 5:1 (растягивается в 5 раз) |
| Стандартные размеры | 15×30, 20×40, 25×50 мм |
| Расширение | В течение 24-48 часов |
| Паропроницаемость | Высокая (выводит влагу из шва) |

### Герметики

| Полиуретановый | Наружные работы | Высокая адгезия |
