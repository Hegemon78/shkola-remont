# Монтаж — Чистовые материалы

Материалы для финишного монтажа: двери, окна, декоративные элементы.

---

## Межкомнатные двери

### Типы конструкций

| Тип | Особенности |
|-----|-------------|
| Щитовые | Каркас + заполнение, лёгкие |
| Царговые | Разборная конструкция |
| Филёнчатые | Классика, рамка + вставки |
| Погонажные | Рамка + стекло/вставки |

### Покрытия

| Покрытие | Особенности | Цена |
|----------|-------------|------|
| Ламинат | Плёнка, бюджет | € |
| ПВХ | Плёнка, устойчивее | €€ |
| Экошпон | Имитация дерева | €€ |
| Шпон | Натуральный, дорогой | €€€ |
| Эмаль | Крашеные | €€€ |
| Массив | Дерево | €€€€ |

### Комплектующие

| Элемент | Варианты |
|---------|----------|
| Коробка | Телескопическая / обычная |
| Наличники | Плоские / фигурные / телескоп |
| Доборы | Расширение коробки |
| Порог | Алюминий / дерево / без порога |
| Петли | Накладные / врезные / скрытые |
| Ручки | Защёлка / магнит / замок |

**Размеры проёмов:**
- 700×2000 — стандарт
- 800×2000 — расширенный
- 900×2000 — увеличенный
- Коробка занимает ~7-8 см по ширине

## Входные двери

| Параметр | На что смотреть |
|----------|-----------------|
| Толщина металла | 1.5-2 мм минимум |
| Утепление | Минвата/пенополиуретан |
| Уплотнители | 2-3 контура |
| Замки | 2 разных типа, класс 3-4 |
| Петли | Скрытые или с шариком |

## Окна

### Типы профилей

| Класс | Камеры | Применение |
|-------|--------|------------|
| A-класс | 3-5 | Стандарт жилья |
| B-класс | 3 | Бюджет |
| Премиум | 6-7 | Тихие улицы, север |

### Стеклопакеты

| Тип | Формула | Применение |
|-----|---------|------------|
| Однокамерный | 4-16-4 | Тёплый климат |
| Двухкамерный | 4-10-4-10-4 | Стандарт |
| Энергосберегающий | И-стекло | Экономия тепла |
| Мультифункциональный | Solar | Защита от солнца |

### Комплектующие

| Элемент | Варианты |
|---------|----------|
| Откосы | Пластик / штукатурка / сэндвич |
| Подоконник | Пластик / камень / дерево |
| Отливы | Металл / пластик |
| Фурнитура | Поворотно-откидная / раздвижная |
| Москитные сетки | Рамочная / рулонная |
