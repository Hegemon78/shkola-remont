# Электрика — Чистовые материалы

Материалы для финишного этапа электромонтажа.

---

## Розетки и выключатели

### Механизмы

| Тип | Применение |
|-----|------------|
| Розетка 16А с заземлением | Стандарт для всех помещений |
| Розетка 16А без заземления | ❌ Не использовать |
| Розетка 32А | Электроплита (редко, обычно клеммы) |
| Выключатель 1-клавишный | Одна группа света |
| Выключатель 2-клавишный | Две группы света |
| Проходной выключатель | Управление из 2 точек |
| Перекрёстный выключатель | Управление из 3+ точек |
| Диммер | Регулировка яркости |

**Нюансы:**
- Глубокие подрозетники для диммеров и проходных
- Проверять совместимость диммеров с LED-лампами
- Влагозащита IP44+ для ванной

---

### Рамки

| Материал | Особенности |
|----------|-------------|
| Пластик | Бюджет, большой выбор цветов |
| Стекло | Премиум, легко царапается |
| Металл | Премиум, прочные |
| Дерево | Премиум, для интерьеров |

**Серии (примеры):**
- Бюджет: Schneider Sedna, Legrand Valena Life
- Средний: ABB Zenit, Legrand Celiane
- Премиум: Gira, Jung, Berker

## Специальные розетки

| Тип | Применение |
|-----|------------|
| USB A+C | Зарядка гаджетов |

| Аудио | Встроенные колонки |

---

## Освещение (установка)

### Точечные светильники

| Тип | Применение | Размер отверстия |
|-----|------------|------------------|
| GU10 | Стандарт, 220V | 70-90 мм |
| GU5.3 | 12V, нужен трансформатор | 70-90 мм |
| GX53 | Тонкие, для низких потолков | 90 мм |

**Нюансы:**
- Для натяжных потолков — только LED (не греются)
- Трансформаторы для GU5.3 размещать в доступном месте
- Учитывать глубину запотолочного пространства

---

### Люстры и светильники

**Крепление:**
- Крюк — для тяжёлых люстр
- Планка — стандартное крепление
- Крестовина — для больших светильников

**Провод:**
- ШВВП 2×0.75 — для светильников до 1 кВт
- ПВС 3×1.5 — если нужно заземление
