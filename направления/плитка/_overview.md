# Плиточные работы

> Укладка плитки, мозаики.

---

## Описание

Облицовка поверхностей керамической плиткой, керамогранитом, мозаикой.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Подготовка основания | Проверка ровности, грунтовка |
| Гидроизоляция | Мокрые зоны (ЧЕРНОВОЙ ЭТАП!) |
| Раскладка плитки | Планирование рисунка |
| Укладка на пол | Сначала пол (много грязи) |
| Укладка на стены | После пола |
| Укладка фартука | Кухонный фартук |
| Укладка мозаики | Мелкая плитка на сетке |
| Подрезка | Резка под размер |
| Затирка швов | Заполнение швов |
| Герметизация | Силикон в углах, примыканиях |

---

## Порядок укладки

1. **Пол** — сначала (грязная работа, ходим по нему)
2. **Стены** — после пола

---

## Требования к рабочему

- Умение работать с плиткой
- Знание клеевых составов
- Работа с плиткорезом
- Понимание раскладки, рисунка

---

## Материалы

- Плитка, керамогранит, мозаика
- Плиточный клей
- Затирка
- Крестики, СВП
- Грунтовка
- Герметик

---

## Инструменты

- Плиткорез (ручной, электрический)
- Болгарка с диском
- Гребёнка (шпатель зубчатый)
- Уровень
- Резиновый шпатель (для затирки)

---

## Связанные направления

- Монтаж (гидроизоляция — черновой этап)
- Сантехника (после плитки — чистовая)
- Полы черновые (стяжка до плитки)
