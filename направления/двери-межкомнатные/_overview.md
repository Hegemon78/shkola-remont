# Двери межкомнатные

> Установка межкомнатных дверей.

---

## Описание

Монтаж межкомнатных дверных блоков: коробки, полотна, наличники, фурнитура.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Замер проёма | Проверка размеров |
| Сборка коробки | Монтаж дверной коробки |
| Установка коробки | Крепление в проём |
| Запенивание | Монтажная пена |
| Установка полотна | Навешивание двери |
| Регулировка | Подгонка, выравнивание |
| Установка наличников | Декоративные планки |
| Установка фурнитуры | Ручки, замки, петли |
| Установка порога | Если требуется |

---

## Когда устанавливать

- **После финишного покрытия полов**
- После покраски стен

---

## Требования к рабочему

- Умение работать с уровнем
- Аккуратность при запенивании
- Знание типов дверей и фурнитуры
- Точность в установке

---

## Материалы

- Дверной блок (коробка + полотно)
- Наличники
- Фурнитура (ручки, замки, петли)
- Монтажная пена

---

## Инструменты

- Шуруповёрт
- Уровень
- Пистолет для пены
- Стамеска
- Ножовка / торцовка

---

## Связанные направления

- Монтаж (дверные проёмы)
- Полы чистовые (после полов)
