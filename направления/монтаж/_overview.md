# Монтаж

> Стены, перегородки, потолки, каркасы.

---

## Описание

Возведение и монтаж конструкций: перегородки из ГКЛ, каркасные системы, подвесные потолки, обшивка стен.

---

## Процессы направления

### Черновой монтаж

| Процесс | Описание |
|---------|----------|
| Разметка | Планирование расположения перегородок |
| Возведение перегородок (кладка) | Кирпич, газоблок, пазогребень |
| Монтаж каркаса перегородок | Металлический профиль для ГКЛ |
| Дверные проёмы | Усиление, закладные |
| Закладные | Под мебель, технику, ТВ |
| Звукоизоляция | Укладка изолятора в каркас |
| Обшивка ГКЛ | Монтаж гипсокартона |
| Монтаж потолка | Каркас подвесного потолка |
| Гидроизоляция | Мокрые зоны (перед плиткой) |

### Чистовой монтаж

| Процесс | Описание |
|---------|----------|
| Короба | Декоративные короба для труб, вентиляции |
| Ниши | Ниши под технику, декор |
| Потолочные элементы | Многоуровневые потолки |

---

## Требования к рабочему

- Умение работать с профилями и ГКЛ
- Знание типов крепежа
- Понимание геометрии и уровней
- Работа с кладкой (кирпич, блоки)

---

## Материалы

- Профиль (ПН, ПС, ПП, ПНП)
- Гипсокартон (обычный, влагостойкий, огнестойкий)
- Кирпич, газоблок, пазогребень
- Звукоизоляция (минвата)
- Гидроизоляция
- Крепёж (саморезы, дюбели)

---

## Инструменты

- Перфоратор
- Шуруповёрт
- Лазерный уровень
- Просекатель
- Ножницы по металлу

---

## Связанные направления

- Электрика (закладные, провода в перегородках)
- Сантехника (закладные под трубы)
- Изоляция (звуко-, теплоизоляция)
- Штукатурка (после обшивки)
