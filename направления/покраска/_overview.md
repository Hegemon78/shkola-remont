# Покраска

> Малярные работы — окрашивание стен и потолков.

---

## Описание

Нанесение лакокрасочных покрытий на подготовленные поверхности.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Подготовка к покраске | Укрывка, заклейка |
| Грунтовка | Перед покраской |
| Покраска потолка | Кистью, валиком, краскопультом |
| Покраска стен | Кистью, валиком, краскопультом |
| Покраска откосов | Оконные, дверные |
| Покраска радиаторов | Батареи, трубы |
| Декоративная покраска | Фактурные техники |

---

## Варианты техники

| Способ | Когда использовать |
|--------|-------------------|
| Валик/кисть | Потолок → Стены |
| Краскопульт (компрессор) | Стены → Потолок |

---

## Требования к рабочему

- Умение работать с краской
- Знание типов красок
- Аккуратность
- Для краскопульта — работа с оборудованием

---

## Материалы

- Краска (водоэмульсионная, латексная, акриловая)
- Грунтовка
- Малярный скотч
- Укрывная плёнка

---

## Инструменты

- Валики (разные)
- Кисти
- Краскопульт
- Лоток для краски
- Телескопическая ручка

---

## Связанные направления

- Подготовка стен (перед покраской)
- Полы чистовые (полы укрываются)
