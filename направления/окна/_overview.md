# Окна

> Установка окон, отделка откосов.

---

## Описание

Монтаж оконных конструкций и отделка оконных откосов.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Демонтаж старых окон | Если требуется |
| Подготовка проёма | Очистка, выравнивание |
| Установка окна | Монтаж рамы |
| Крепление | Анкера, пластины |
| Запенивание | Монтажная пена |
| Установка подоконника | Монтаж подоконника |
| Установка отлива | Внешний отлив |
| Откосы | Штукатурные, пластиковые, ГКЛ |
| Утепление откосов | Теплоизоляция |

---

## Когда устанавливать

- **Лучше после штукатурки** — для вывода в уровень с откосами
- Подоконник — **до покраски** (для примыкания)

---

## Требования к рабочему

- Знание оконных конструкций
- Умение работать с уровнем
- Работа с монтажной пеной
- Отделка откосов

---

## Материалы

- Окна (ПВХ, дерево, алюминий)
- Подоконник
- Отлив
- Монтажная пена
- Анкера, пластины

---

## Инструменты

- Перфоратор
- Уровень
- Пистолет для пены
- Шуруповёрт

---

## Связанные направления

- Штукатурка (откосы)
- Подготовка стен (откосы под покраску)
- Покраска (откосы)
