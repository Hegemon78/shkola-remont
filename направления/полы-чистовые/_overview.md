# Чистовые полы

> Паркет, ламинат, плитка, ковролин.

---

## Описание

Укладка финишных напольных покрытий.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Подготовка основания | Проверка ровности, влажности |
| Укладка подложки | Под ламинат, паркет |
| Укладка ламината | Плавающий способ |
| Укладка паркетной доски | Плавающий или на клей |
| Укладка массива | На клей |
| Укладка инженерной доски | На клей |
| Укладка ковролина | На клей или свободно |
| Укладка ПВХ-плитки | Замковая или на клей |
| Укладка линолеума | Свободно или на клей |
| Укрывка | Защита на время покраски стен |
| Шлифовка (паркет) | При необходимости |
| Лакировка/масло | Финишное покрытие |

---

## Порядок работ

1. Укладка покрытия
2. **Укрывка** (защита)
3. Покраска стен
4. Снятие укрывки
5. Плинтусы

---

## Требования к рабочему

- Знание типов покрытий
- Умение работать с геометрией
- Аккуратность в подрезке
- Работа с замками, клеем

---

## Материалы

- Ламинат, паркет, массив
- Подложка
- Клей для паркета
- Плинтус, порожки

---

## Инструменты

- Электролобзик
- Торцовочная пила
- Добойник
- Скоба для подбивки
- Угольник
- Рулетка

---

## Связанные направления

- Полы черновые (после стяжки)
- Покраска (полы укрываются)
- Плитка (стыковка покрытий)
