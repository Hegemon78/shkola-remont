# Климат

> Вентиляция, кондиционирование.

---

## Описание

Монтаж систем вентиляции и кондиционирования для обеспечения комфортного микроклимата.

---

## Процессы направления

### Черновая климат

| Процесс | Описание |
|---------|----------|
| Разметка трасс | Планирование расположения |
| Прокладка трасс кондиционера | Медные трубки, дренаж, кабель |
| Установка наружного блока | Монтаж внешнего блока |
| Монтаж воздуховодов | Вентиляционные каналы |
| Установка вентиляторов | Канальные вентиляторы |
| Приточная вентиляция | Бризеры, приточные клапаны |

### Чистовая климат

| Процесс | Описание |
|---------|----------|
| Установка внутреннего блока | Монтаж сплит-системы |
| Установка решёток | Вентиляционные решётки |
| Пусконаладка | Заправка, проверка |

---

## Требования к рабочему

- Понимание принципов вентиляции
- Знание типов кондиционеров
- Умение работать с медными трубами
- Работа с вакуумным насосом

---

## Материалы

- Медные трубы
- Теплоизоляция
- Дренажные трубки
- Воздуховоды
- Решётки

---

## Инструменты

- Труборез для меди
- Вальцовка
- Вакуумный насос
- Манометрический коллектор

---

## Связанные направления

- Электрика (подключение блоков)
- Монтаж (короба, ниши)
