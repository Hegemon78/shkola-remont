# Штукатурные работы

> Выравнивание стен и потолков штукатуркой.

---

## Описание

Нанесение штукатурных составов для выравнивания поверхностей. Ручная и механизированная штукатурка.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Подготовка поверхности | Очистка, грунтовка |
| Установка маяков | Направляющие для ровного слоя |
| Штукатурка ручная | Нанесение вручную |
| Штукатурка механизированная | Нанесение станцией |
| Штукатурка откосов | Оконные, дверные откосы |
| Формирование углов | Установка уголков |
| Удаление маяков | Извлечение, заделка |
| Затирка | Финишная обработка |

---

## Требования к рабочему

- Умение работать с маяками
- Знание штукатурных смесей
- Работа с правилом
- Для мех. штукатурки — работа со станцией

---

## Материалы

- Штукатурка (гипсовая, цементная)
- Грунтовка
- Маяки
- Уголки (перфорированные)
- Сетка штукатурная

---

## Инструменты

- Правило
- Шпатели, мастерок
- Тёрка, полутёрок
- Лазерный уровень
- Штукатурная станция (для мех.)

---

## Связанные направления

- Подготовка стен (после штукатурки)
- Монтаж (после обшивки ГКЛ)
