# Сантехника

> Водоснабжение, канализация, отопление.

---

## Описание

Монтаж и обслуживание систем водоснабжения, водоотведения и отопления.

---

## Процессы направления

### Черновая сантехника

| Процесс | Описание |
|---------|----------|
| Разметка трасс | Планирование расположения труб |
| Штробление под трубы | Каналы в стенах |
| Разводка водоснабжения | ХВС, ГВС — трубы ППР, металлопластик |
| Разводка канализации | Трубы 50мм, 110мм |
| Монтаж в полу | Трубы, трапы |
| Опрессовка | Проверка герметичности (обязательно!) |
| Сантехнический шкаф | Счётчики, фильтры, коллекторы |
| Отопление — трубы | Разводка радиаторного отопления |
| Коллекторный шкаф | Коллекторы тёплого пола, отопления |

### Чистовая сантехника

| Процесс | Описание |
|---------|----------|
| Установка унитаза | Монтаж, подключение |
| Установка раковины | Монтаж раковины, тумбы |
| Установка смесителей | Краны, смесители |
| Установка ванны/душа | Ванна, душевая кабина, поддон |
| Полотенцесушитель | Монтаж, подключение |
| Установка радиаторов | Финишная установка батарей |
| Аксессуары | Держатели, крючки, зеркала |

---

## Требования к рабочему

- Знание типов труб и соединений
- Понимание гидравлики
- Умение паять ППР
- Работа с пресс-фитингами

---

## Материалы

- Трубы (ППР, металлопластик, медь)
- Фитинги
- Канализационные трубы
- Запорная арматура
- Уплотнители

---

## Инструменты

- Паяльник для ППР
- Труборез
- Пресс-инструмент
- Ключи

---

## Связанные направления

- Электрика (подключение бойлера, насосов)
- Монтаж (штробление, закладные)
- Плитка (установка после плитки)
- Полы черновые (трубы в стяжке)
