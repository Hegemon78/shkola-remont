# Двери входные

> Установка входных дверей.

---

## Описание

Монтаж входных дверей — металлических, деревянных.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Демонтаж старой двери | Если требуется |
| Подготовка проёма | Очистка, выравнивание |
| Установка коробки | Крепление рамы |
| Крепление | Анкера |
| Запенивание | Монтажная пена |
| Установка полотна | Навешивание двери |
| Регулировка | Подгонка, проверка замков |
| Отделка откосов | Внутренние откосы |
| Установка доборов | Если проём широкий |
| Установка наличников | Декоративные планки |

---

## Когда устанавливать

- **После штукатурки, стяжки** и всех грязных/пыльных работ
- До чистовой отделки входной зоны

---

## Требования к рабочему

- Опыт работы с металлическими дверями
- Работа с анкерами
- Знание запорных механизмов
- Отделка откосов

---

## Материалы

- Входная дверь
- Анкера
- Монтажная пена
- Доборы, наличники

---

## Инструменты

- Перфоратор
- Уровень
- Болгарка
- Пистолет для пены

---

## Связанные направления

- Штукатурка (откосы)
- Подготовка стен (откосы под покраску)
