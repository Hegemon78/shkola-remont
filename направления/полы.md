# Полы

Стяжка, укладка покрытий.

## Описание

Направление включает работы по подготовке и устройству полов: стяжки, наливные полы, укладка ламината, паркета, плитки.

## Этапы применения

- [ ] Черновая (стяжка, выравнивание)
- [ ] Чистовая (укладка покрытий)
