# Звукоизоляция / Утепление

> Изоляционные работы для звука и тепла.

---

## Описание

Монтаж звукоизоляционных и теплоизоляционных материалов в стенах, полах, потолках.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Звукоизоляция стен | Каркас + изолятор + обшивка |
| Звукоизоляция пола | Плавающая стяжка |
| Звукоизоляция потолка | Подвесной потолок с изолятором |
| Звукоизоляция перегородок | Заполнение каркаса |
| Утепление стен | Внутреннее утепление |
| Утепление пола | Под стяжку |
| Утепление откосов | Оконные, дверные откосы |

---

## Требования к рабочему

- Понимание принципов звукоизоляции
- Знание теплоизоляционных материалов
- Умение работать с каркасами

---

## Материалы

- Минеральная вата
- Звукоизоляционные панели
- ЭППС, пенополистирол
- Виброподвесы
- Демпферная лента

---

## Инструменты

- Нож для утеплителя
- Шуруповёрт
- Строительный степлер

---

## Связанные направления

- Монтаж (каркасы перегородок, потолков)
- Полы черновые (под стяжку)
