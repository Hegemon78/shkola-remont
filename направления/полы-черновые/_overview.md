# Черновые полы

> Стяжка, выравнивание, тёплый пол.

---

## Описание

Подготовка основания пола: стяжки, выравнивание, монтаж систем тёплого пола.

---

## Процессы направления

| Процесс | Описание |
|---------|----------|
| Подготовка основания | Очистка, ремонт трещин |
| Гидроизоляция | В мокрых зонах |
| Демпферная лента | По периметру |
| Укладка утеплителя | ЭППС, пенополистирол |
| Армирование | Сетка для стяжки |
| Монтаж тёплого пола | Электрический кабель или водяные трубы |
| Установка маяков | Направляющие |
| Стяжка (мокрая) | ЦПС, полусухая |
| Стяжка (сухая) | ГВЛ, керамзит |
| Наливной пол | Финишное выравнивание |

---

## Порядок работ

1. Коммуникации в полу (электрика, сантехника)
2. Гидроизоляция (если нужна)
3. Утеплитель (если нужен)
4. Армирование
5. Тёплый пол
6. Стяжка
7. Высыхание
8. Наливной пол (если нужен)

---

## Требования к рабочему

- Понимание типов стяжек
- Умение работать с уровнем, маяками
- Знание тёплого пола
- Работа с правилом

---

## Материалы

- ЦПС (цементно-песчаная смесь)
- Наливной пол
- Армирующая сетка
- Демпферная лента
- ЭППС
- Тёплый пол (кабель, маты, трубы)

---

## Инструменты

- Правило
- Лазерный уровень
- Миксер
- Игольчатый валик (для наливного)

---

## Связанные направления

- Электрика (тёплый пол, провода в стяжке)
- Сантехника (трубы в полу)
- Изоляция (утепление)
- Полы чистовые (после высыхания)
