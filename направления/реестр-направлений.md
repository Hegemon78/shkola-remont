# Направления работ

Основные специализации в ремонте и строительстве.

---

## Логика связей

```
НАПРАВЛЕНИЕ (что)
    │
    ├── содержит ПРОЦЕССЫ (конкретные работы)
    │       │
    │       └── каждый процесс привязан к ЭТАПУ (когда)
    │       │
    │       └── каждый процесс содержит ТЕХНОЛОГИИ (как)
    │               │
    │               └── каждая технология содержит ШАГИ + ЧЕК-ЛИСТЫ
```

## Структура файлов

```
/направления/
    реестр-направлений.md           ← этот файл
    /культура-производства/
        D00-культура-производства.md  ← описание + процессы
    /электрика/
        D02-электрика.md              ← описание + процессы
    ...

/процессы/
    /электрика-черновая/
        штробление.md
        прокладка-кабеля.md
    /электрика-чистовая/

    ...

/технологии/
    штробление-штроборезом.md
    штробление-перфоратором.md
    ...
```
