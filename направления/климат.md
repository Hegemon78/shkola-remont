# Климат

Вентиляция, кондиционирование.

## Описание

Направление включает работы по монтажу систем вентиляции, кондиционирования и климат-контроля.

## Этапы применения

- [ ] Черновая (закладка трасс, воздуховодов)
- [ ] Чистовая (установка блоков, решёток)
