# Этапы ремонта — Общая логика

---

## Базовая последовательность

```
1. Подготовка объекта (культура производства) — постоянно
2. Демонтаж
3. Черновые работы
4. Чистовые работы
5. Финиш / Сдача
6. Мебель (внешний этап — координация)
```

## Принципы

1. **Культура производства** — не отдельный этап, а постоянное правило
2. **Укрывочные работы** — на каждом этапе по необходимости
3. **Параллельность** — некоторые работы могут идти одновременно в разных зонах
4. **Зависимости** — некоторые работы строго после других (см. схемы в файлах)
5. **Вариативность** — некоторые этапы могут меняться местами (указано в файлах)
